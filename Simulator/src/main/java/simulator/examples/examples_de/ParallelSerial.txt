In diesem Beispielmodell wird eine serielle Verarbeitung einer parallelen Verarbeitung gegenübergestellt. Insgesamt wird jedes eintreffende Objekt in jedem der beiden Fälle von drei Stationen geprüft. An jeder Station wird ein gewisser Anteil der Objekte aussortiert. Bei der seriellen Verarbeitung führt dies dazu, dass die Arbeitslast an der zweiten und der dritten Station niediger ausfällt. Im parallelen Fall muss jedes Objekt von jeder Station verarbeitet werden (was zu einer höheren Arbeitslast führt). Dafür fallen die Durchlaufzeiten kürzer aus.
