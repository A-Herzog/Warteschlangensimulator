In diesem Beispielmodell durchläuft jeder Kunde 10x eine Verzögerungsstation, an der eine exponentiell verteilte Wartezeit auftritt. Dargestellt wird das Histogramm der gesamten Wartezeiten der Kunden, d.h. das Histogramm von Summen aus jeweils 10 exponentiell verteilten Zufallszahlen. Diese Summenverteilung nähert sich dabei einer Normalverteilung an. Dies ist die Aussage des zentralen Grenzwertsatzes.
