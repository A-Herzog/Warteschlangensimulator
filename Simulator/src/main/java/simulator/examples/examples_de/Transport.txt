In diesem Beispielmodell wird eine Möglichkeit zur Modellierung von Transporten von Werkstücken zwischen verschiedenen Stationen dargestellt. Transporte stellen eine Möglichkeit zum Transfer von Objekten in dem betrachteten Netzwerk dar, die keine Bewegung entlang einer Kante benötigt. Option können bestimmte Ressourcen für einen Transport benötigt werden und es können Zeitdauern für Transporte definiert werden.
