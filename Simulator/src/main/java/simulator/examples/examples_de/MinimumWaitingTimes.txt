Soll die Auslastungen zwischen verschieden teuren Bedienstationen möglichst stark ausdifferenziert werden, so kann dies über entsprechende Zuweisungsstrategien erfolgen. Reicht dies nicht aus, so können zusätzlich Mindestwartezeiten, bevor die jeweils teurere Station betreten werden darf, eingesetzt werden. Im Gegensatz zu reinen Zuweisungsstrategien verschlechtern diese jedoch die kundenspezifischen Kenngrößen des Systems.
