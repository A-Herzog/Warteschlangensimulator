Puffer vor Stationen können dafür sorgen, dass eine Station auch bei einem kurzfristigen Ausfall der Vorgängerstation weiterarbeiten kann. Auf der anderen Seite können hohe Bestände dazu führen, dass, um die FIFO-Bedienreihenfolge einzuhalten, zusätzlicher Aufwand für das Umstapeln notwendig wird, d.h. dass sich die Bediendauern verlängern. In diesem Beispielmodell können die Effekte von kleinen oder großen Puffern analysiert werden.
