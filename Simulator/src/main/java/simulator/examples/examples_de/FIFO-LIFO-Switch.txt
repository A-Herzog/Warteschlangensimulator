In diesem Modell wird untersucht, wie sich die Bedienreihenfolge (FIFO oder LIFO) auf die Kenngrößen eines Systems auswirkt: Im FIFO-Fall ist die Streuung der Wartezeiten geringer als im LIFO-Fall. Allerdings ist das Modell so konfiguriert, dass die Bedienzeiten im FIFO-Modus mit steigender Warteschlangenlänge ansteigen. Daher wird ab einem bestimmten (konfigurierbaren) Punkt in den schnelleren LIFO-Modus umgeschaltet.
