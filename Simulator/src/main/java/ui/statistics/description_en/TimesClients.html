This page shows how long the clients have been in the system. The total time from arrival to departure from the system is called residence time. The average residence time is called E[V]. The residence time is made up of waiting, transfer and service time. These are designated by E[W], E[T] and E[S].

The data is displayed globally, across all client types and individually per client type. (If only one client type exists in the system, this subdivision is omitted).

For all time entries average values, standard deviations, variances, coefficients of variation and quantiles are displayed.
