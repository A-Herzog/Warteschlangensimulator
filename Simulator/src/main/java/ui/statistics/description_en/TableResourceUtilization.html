Meaning of the columns in this table:

  • Resource: Operator group to which the respective line refers.
  • Number of operators: Number of operators in the respective group, if this is determined by a fixed number (and not, for example, by a shift schedule).
  • rho: If the number of operators in this group is a fixed value during simulation, rho indicates the average utilization of the operator group.
  • Average busy: Average absolute number of working operators in this group.
  • Std: Standard deviation of the absolute number of working operators in this group.
  • Var: Variance of the absolute number of working operators in this group.
  • CV: Coefficient of variation of the absolute number of working operators in this group.
  • Sk: Skewness of the absolute number of working operators in this group.
  • Kurt: Excess kurtosis of the absolute number of working operators in this group.
  • Min: Minimum absolute number of working operators during the simulation in this group.
  • Max: Maximum absolute number of working operators during the simulation in this group.
  • Idle share (all operators): Outputs the part of time during which not a single operator in the group was working.
  • Full busy share: If the number of operators in this group is a fixed value during the simulation, this cell indicates the part of time during which all operators were busy.

The value rho indicates the ratio between busy and total available operators. If the number of available operators in the group is either variable (defined by a shift plan) or if the group consists of an infinite number of operators, no value can be determined for rho. The same applies to the full load share: If there is no fixed number of available operators, no time share in which this number was busy can be output.

Values that refer to the absolute number of busy operators, however, are always available.
