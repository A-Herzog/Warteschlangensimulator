This page shows the data collected at Statistics stations.

Meaning of the columns in this table:

  • User-defined statistics: Identifier set in the statistics station for the parameter to be recorded to which the respective row refers.
  • Mean: Average value of the recorded parameter
  • Standard deviation: Standard deviation of the recorded parameter
  • Var: Variance of the recorded parameter
  • CV: Coefficient of variation of the recorded parameter
  • Sk: Skewness of the recorded parameter
  • Kurt: Excess kurtosis of the recorded parameter
  • Minimum: Minimum value of the recorded parameter
  • Maximum: Maximum value of the recorded parameter
  • 10% quantile: 10% quantile of the value of the recorded parameter. 10% of the recorded values were at most as large as shown in this cell.
  • 25% quantile: 25% quantile of the value of the recorded parameter. 25% of the recorded values were at most as large as shown in this cell.
  • 50% quantile: 50% quantile of the value of the recorded parameter. 50% of the recorded values were at most as large as shown in this cell. This value is also the median of the recorded parameter.
  • 75% quantile: 75% quantile of the value of the recorded parameter. 75% of the recorded values were at most as large as shown in this cell.
  • 90% quantile: 90% quantile of the value of the recorded parameter. 90% of the recorded values were at most as large as shown in this cell.
