Meaning of the columns in this table:

  • Resource: Transporter group to which the respective row refers.
  • Number of operators of this type: Number of transporters in the respective group.
  • Share in down time: Indicates the average share of transports in the transporter group who are in down time.
  • Mean: Average absolute number of transporters in this group in down time.
  • Std: Standard deviation of the absolute number of transporters in down time in this group.
  • Var: Variance of the absolute number of transporters in down time in this group.
  • CV: Coefficient of variation of the absolute number of transporters in down time in this group.
  • Sk: Skewness of the absolute number of transporters in down time in this group.
  • Kurt: Excess kurtosis of the absolute number of transporters in down time in this group.
  • Min: Minimum absolute number of transporters in down time during the simulation in this group.
  • Max: Maximum absolute number of transporters in down time during the simulation in this group.
