This table shows a frequency distribution of one or more temporal quantities. In the first column of each row, the seconds value to which the row refers is displayed. The other columns show how often the respective seconds value for the respective quantity occurred during the simulation (absolute and relative).

The frequency values displayed in the last row refer to the second value displayed in the first column and all larger values. This means that the frequency values in the last row collect all values that are larger than the seconds values specified in the previous row.
