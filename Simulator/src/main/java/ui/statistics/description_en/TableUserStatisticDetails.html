This table shows a frequency distribution of the values of the user-defined statistics stations. In the first column of each row, the status value to which the row refers is given. The other columns show how often this value was observed (absolute and relative).

The relative frequencies displayed in the last row refer to the state value displayed in the first column and all larger values. This means that the relative frequencies in the last row form the totals for all state values that are greater than the values specified previously.
