Meaning of the columns in this table:

  • Resource: Transporter group to which the respective row refers.
  • Number of operators of this type: Number of transporters in the respective group.
  • rho: Indicates the average utilization of the transporter group.
  • Average number of driving transporters: Average absolute number of driving transporters in this group.
  • Std: Standard deviation of the absolute number of driving transporters in this group.
  • Var: Variance of the absolute number of driving transporters in this group.
  • CV: Coefficient of variation of the absolute number of driving transporters in this group.
  • Sk: Skewness of the absolute number of driving transporters in this group.
  • Min: Minimum absolute number of driving transporters during the simulation in this group.
  • Max: Maximum absolute number of driving transporters during the simulation in this group.
  • Idle share (all transporters): Outputs the part of time during which not a single transporter in the group was driving.
  • Full busy share: Outputs the part of time during which all transporters of the group were driving simultaneously.
