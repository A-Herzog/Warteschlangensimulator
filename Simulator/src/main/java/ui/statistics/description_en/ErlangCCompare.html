Erlang-C formula

By using the Erlang-C formula the relevant performance indicators of simple queueing models like average waiting, service and residence times (E[W], E[S] and E[V]), the number of clients in the system, in the queue and in the service process (E[N], E[NQ] and E[NS]) as well as the utilization of the operators ρ can be calculated exactly. In this context "simple" means the model needs to have the following properties:

  • The inter-arrival times are exponential distributed.
  • The service times are exponential distributed.
  • There is a fixed number of operators.
  • There are not other special properties (like different client types, batch arrival, batch service, branching between stations, ...).

The classical Erlang-C formula cannot represent clients impatience. But the extended Erlang-C formula can represent impatience, so the following property may also prevail in a simple model:

  • Optionally, clients are only willing to wait for a limited time. If this is the case, it waiting time tolerance has to be exponential distributed.

A corresponding model is then called M/M/c or M/M/c/ infinity+M model.

If the queueing model to be simulated is simple enough so that the simulator can establish a clear relationship between the simulated model and the Erlang-C model, it compares the simulation results with the Erlang-C values. If the simulated model fulfills all above mentioned conditions to an Erlang-C model, the differences between simulation results and theoretical results are very small (typically <1%). If there are properties in the simulated model that are not covered by the Erlang-C formula, significant deviations between the formula results and the simulation results can be expected.

Allen-Cunneen approximation formula

The Allen-Cunneen approximation formula allows a relaxation of the restrictions defined in the Erlang-C model: Instead of exponentially distributed inter-arrival and service times, any distributions characterized by their expected value and coefficient of variation are now allowed. In addition, batch arrivals and batch service are allowed. (The batch sizes are denoted by b(I) and b(S).) For this, the absolute accuracy has to be given up, i.e. even with a model which complies with all specifications of the Allen-Cunneen approximation formula, differences between the approximation formula results and the simulation reality have to be expected.
