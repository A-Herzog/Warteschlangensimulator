Bedeutung der Spalten in dieser Tabelle:

  • Resource: Operator group to which the respective line refers.
  • Number of operators: Number of operators in the respective group, if this is determined by a fixed number (and not, for example, by a shift schedule).
  • Share in down time: If the number of operators in this group is a fixed value during simulation, this cell indicates the average share of operators in the operator group who are in down time.
  • Mean: Average absolute number of operators in this group in down time.
  • Std: Standard deviation of the absolute number of operators in down time in this group.
  • Var: Variance of the absolute number of operators in down time in this group.
  • CV: Coefficient of variation of the absolute number of operators in down time in this group.
  • Sk: Skewness of the absolute number of operators in down time in this group.
  • Kurt: Excess kurtosis of the absolute number of operators in down time in this group.
  • Min: Minimum absolute number of operators in down time during the simulation in this group.
  • Max: Maximum absolute number of operators in down time during the simulation in this group.

The value share in down time indicates the ratio between the operators in down time and the total number of available operators. If the number of available operators in the group is either variable (defined by a shift plan) or if the group consists of an infinite number of operators, no share can be determined.

Values that refer to the absolute number of operators in down time, however, are always available.
