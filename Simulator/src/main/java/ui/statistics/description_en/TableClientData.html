Meaning of the columns in this table:

  • Client data field: Number of the client data field to which the respective row refers.
  • E: Mean of the client data field
  • Std: Standard deviation of the client data field
  • Var: Variance of the client data field
  • CV: Coefficient of variation of the client data field
  • Sk: Skewness of the client data field
  • Kurt: Excess kurtosis of the client data field
  • Min: Minimum value of the client data field
  • Max: Maximum value of the client data field
  • 10% quantile: 10% quantile of the value of the client data field. 10% of the recorded values were at most as large as shown in this cell.
  • 25% quantile: 25% quantile of the value of the client data field. 25% of the recorded values were at most as large as shown in this cell.
  • 50% quantile: 50% quantile of the value of the client data field. 50% of the recorded values were at most as large as shown in this cell. This value is also the median of the client data field values.
  • 75% quantile: 75% quantile of the value of the client data field. 75% of the recorded values were at most as large as shown in this cell.
  • 90% quantile: 90% quantile of the value of the client data field. 90% of the recorded values were at most as large as shown in this cell.

The values of the data fields of the individual clients are recorded when the clients are leaving the system.
