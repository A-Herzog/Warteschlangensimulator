Der Flussgrad gibt das Verhältnis von Verweilzeit zu Bedienzeit ein. Der Flussgrad ist folglich immer größer oder gleich 1. Kleine Werte bedeuten, dass im Verhältnis zur Bedienzeit nur wenig Warte- und Transferzeiten angefallen sind. Große Werte deuten auf verhältnismäßig lange Warte- und Transferzeiten hin.
