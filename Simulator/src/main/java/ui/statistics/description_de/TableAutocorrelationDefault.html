Autokorrelation

Häufig ist es wünschenswert, nicht nur Kenngrößen wie den Mittelwert über die während einer Simulation erfassten Messwerte (z.B. für die mittleren Wartezeiten der Kunden) auszugeben, sondern auch bewerten zu können, mit einer wie hohen Sicherheit sich der tatsächliche, langfristige Mittelwert dieser Größe mit welcher Wahrscheinlichkeit wie nah an dem experimentell ermittelten Wert befindet. Dies leisten die sogenannten Konfidenzintervalle.

In die Formel zur Berechnung eines Konfidenzintervalls geht der gemessene Mittelwert, die gemessene Streuung der Werte (Standardabweichung) und die Anzahl an unabhängigen Messwerten ein. Die ersten beiden Kenngrößen werden für alle relevanten Eigenschaften während der Simulation erfasst. Die Festlegung, wie viele unabhängige Messwerte es gab, ist jedoch weniger einfach: Musste ein Kunde lange warten, so ist die Wahrscheinlichkeit hoch, dass auch der nächste Kunde recht lange wird warten müssen. Umgekehrt bedeutet eine kurze Wartezeit für einen Kunden, dass der unmittelbar folgende Kunde ebenfalls wird nicht sehr lange warten müssen. Diese Abhängigkeit zwischen den Wartezeiten der Kunden wird die Autokorrelation genannt. Für die Bestimmung von Konfidenzintervallen ist es daher von Interesse zu wissen, ab welchem Abstand die Rest-Autokorrelation unterhalb eines als akzeptabel einzustufenden Wertes liegt. Die Gesamtanzahl an Ankünften wird dann durch diesen Abstand dividiert, um so eine Anzahl an zumindest näherungsweise unabhängigen Messwerten zu erhalten.
