In diesem Diagramm wird die Auslastung der einzelnen Bedienergruppen dargestellt. Jeder gestapelte Balken steht für eine Bedienergruppe. Die Balken unterteilen sich in Bereiche für die mittlere Anzahl an arbeitenden Bedienern, an Bedienern in Leerlaufzeit und an Bedienern in Pausezeit. Auf der y-Achse sind jeweils die Mittelwerte als absolute Anzahlen an Bedienern aufgetragen.
