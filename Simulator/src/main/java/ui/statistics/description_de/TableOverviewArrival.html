Bedeutung der Spalten in dieser Tabelle:

  • Station oder Kundentyp: Modellstation an der die Kunden eingetroffen sind bzw. Kundentyp dessen Ankunftsdaten in dieser Zeile dargestellt werden.
  • Anzahl: Anzahl an Ankünften an dieser Station während der gesamten Simulation.
  • E[I]: Mittlerer zeitlicher Abstand (in Sekunden) zwischen zwei Ankünften an dieser Station.
  • Std[I]: Standardabweichung der zeitlichen Abstände (in Sekunden) zwischen zwei Ankünften an dieser Station.
  • Var[I]: Varianz der zeitlichen Abstände (in Sekunden) zwischen zwei Ankünften an dieser Station.
  • CV[I]: Variationskoeffizient der zeitlichen Abstände zwischen zwei Ankünften an dieser Station.
  • Sk[I]: Schiefe der zeitlichen Abstände zwischen zwei Ankünften an dieser Station.
  • Min[I]: Minimaler zeitlicher Abstand (in Sekunden) zwischen zwei Ankünften an dieser Station.
  • Max[I]: Maximaler zeitlicher Abstand (in Sekunden) zwischen zwei Ankünften an dieser Station.
  • 10%-Quantil[I]: 10%-Quantil der zeitlichen Abstände zwischen zwei Ankünften an dieser Station. 10% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
  • 25-Quantil[I]: 25%-Quantil der zeitlichen Abstände zwischen zwei Ankünften an dieser Station. 25% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
  • 50%-Quantil[I]: 50%-Quantil der zeitlichen Abstände zwischen zwei Ankünften an dieser Station. 50% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt. Dieser Wert entspricht damit dem Median der Zwischenankunftszeiten.
  • 75%-Quantil[I]: 75%-Quantil der zeitlichen Abstände zwischen zwei Ankünften an dieser Station. 75% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
  • 90%-Quantil[I]: 90%-Quantil der zeitlichen Abstände zwischen zwei Ankünften an dieser Station. 90% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
