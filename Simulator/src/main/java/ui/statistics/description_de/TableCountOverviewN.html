Bei der Anzahl an Kunden an einer Station (bzw. im System) handelt es sich jeweils um die Gesamtanzahl an Kunden an der Station, d.h. es wird nicht zwischen Kunden, die warten, und Kunden, die sich in Bedienung befinden, unterschieden.

Bedeutung der Spalten in dieser Tabelle:

  • Station: Modellstation für die die Anzahl an Kunden in der betreffenden Zeile ausgegeben wird. "System" steht dabei für die Anzahl an Kunden im gesamten System.
  • E[N]: Mittlere Anzahl an Kunden an dieser Station.
  • Std[N]: Standardabweichung der Anzahl an Kunden an dieser Station.
  • Var[N]: Varianz der Anzahl an Kunden an dieser Station.
  • CV[N]: Variationskoeffizient der Anzahl an Kunden an dieser Station.
  • Sk[N]: Schiefe der Anzahl an Kunden an dieser Station.
  • Min[N]: Minimale während der Simulation aufgetretene Anzahl an Kunden an dieser Station.
  • Max[N]: Maximale während der Simulation aufgetretene Anzahl an Kunden an dieser Station.
  • 10%-Quantil[N]: 10%-Quantil der Anzahl an Kunden an dieser Station. Zu 10% der Zeit befanden sich höchstens so viele Kunden wie in dieser Zelle angezeigt an dieser Station.
  • 25-Quantil[N]: 25%-Quantil der Anzahl an Kunden an dieser Station. Zu 25% der Zeit befanden sich höchstens so viele Kunden wie in dieser Zelle angezeigt an dieser Station.
  • 50%-Quantil[N]: 50%-Quantil der Anzahl an Kunden an dieser Station. Zu 50% der Zeit befanden sich höchstens so viele Kunden wie in dieser Zelle angezeigt an dieser Station. Dieser Wert entspricht damit dem Median der Anzahl an Kunden an dieser Station.
  • 75%-Quantil[N]: 75%-Quantil der Anzahl an Kunden an dieser Station. Zu 75% der Zeit befanden sich höchstens so viele Kunden wie in dieser Zelle angezeigt an dieser Station.
  • 90%-Quantil[N]: 90%-Quantil der Anzahl an Kunden an dieser Station. Zu 90% der Zeit befanden sich höchstens so viele Kunden wie in dieser Zelle angezeigt an dieser Station.
