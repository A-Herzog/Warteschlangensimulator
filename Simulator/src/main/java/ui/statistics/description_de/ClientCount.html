Auf dieser Seite wird angezeigt, wie viele Kunden sich jeweils im gesamten System (gesamt und unterschieden nach Kundentypen) und an den einzelnen Stationen befunden haben.

Es wird dabei zwischen der Anzahl an Kunden insgesamt d.h. wartend, in Transport und in Bedienung und nur der Anzahl an wartenden Kunden unterschieden. Für die Gesamtanzahl wird das Kürzel N verwendet, während die wartenden Kunden mit NQ bezeichnet wird.

Für alle Werte werden Mittelwerte, Standardabweichungen, Varianzen, Variationskoeffizienten, Schiefe, Exzess (Wölbung) und Quantile ausgegeben.
