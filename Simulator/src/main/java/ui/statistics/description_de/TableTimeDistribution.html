In dieser Tabelle wird eine Häufigkeitsverteilung einer oder mehrerer zeitlicher Größen dargestellt. In der ersten Spalte jeder Zeile wird jeweils der Sekundenwert, auf den sich die Zeile bezieht, angegeben. In den weiteren Spalten ist angegeben, wie häufig der jeweilige Sekundenwert für die jeweilige Größe während der Simulation aufgetreten ist (absolut und relativ).

Die in der letzten Zeile angezeigten Häufigkeitswerte beziehen sich auf den in der ersten Spalte angezeigten Sekundenwert und alle größeren Werte. D.h. die Häufigkeitswerte in der letzten Zeile sammeln alle Werte auf, die größer als die in zuvor angegebenen Sekundenwerte sind.
