Werden während der Simulation z.B. über eine Variable-Station Datenfelder bei den Kunden mit Werten befüllt (ClientData(index)), so werden diese Felder beim Verlassen der Simulation eines Kunden als statistische Kenngrößen behandelt und es werden die üblichen Kenngrößen (Mittelwerte, Standardabweichungen, Varianzen, Variationskoeffizienten, Schiefe, Exzess (Wölbung) und Quantile) über diese Felder erfasst.
