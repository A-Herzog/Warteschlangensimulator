Bedeutung der Spalten in dieser Tabelle:

  • Ressource: Bedienergruppe auf die sich die jeweilige Zeile bezieht.
  • Anzahl Bediener: Anzahl an Bedienern in der jeweiligen Gruppe, sofern diese durch eine feste Zahl (und nicht z.B. über einen variablen Schichtplan) festgelegt ist.
  • Anteil in Pause: Ist die Anzahl an Bedienern in dieser Gruppe ein während der Simulation fester Wert, so gibt diese Zelle den mittleren Anteil an Bedienern in der Bedienergruppe an, die sich in Pausezeit befinden.
  • Mittelwert: Gibt die mittlere absolute Anzahl an Bedienern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Std: Standardabweichung der absoluten Anzahl an Bedienern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Var: Varianz der absoluten Anzahl an Bedienern in dieser Gruppe an, die sich in Pausezeit befinden.
  • CV: Variationskoeffizient der absoluten Anzahl an Bedienern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Min: Minimale absolute Anzahl an während der Simulation gleichzeitig in Pausezeit befindliche Bediener in dieser Gruppe.
  • Max: Maximale absolute Anzahl an während der Simulation gleichzeitig in Pausezeit befindliche Bediener in dieser Gruppe.

Die Größe Anteil in Pause gibt das Verhältnis von in Pausezeit befindlichen und insgesamt verfügbaren Bedienern an. Ist die Anzahl an verfügbaren Bedienern in der Gruppe entweder variable (Definition über einen Schichtplan) oder aber besteht die Gruppe aus unendlich vielen Bedienern, so kann für den Anteil kein Wert bestimmt werden.

Werte, die sich auf die absolute Anzahl an Bedienern in Pausezeit beziehen, stehen jedoch immer zur Verfügung.
