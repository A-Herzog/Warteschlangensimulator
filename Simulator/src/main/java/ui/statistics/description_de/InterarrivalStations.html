Anzahl an Ankünften

In dieser Rubrik wird angezeigt, wie viele Kunden an den jeweiligen Stationen während der gesamten Simulation eingetroffen sind. Da die Simulation zu einem Zeitpunkt, an dem sich Kunden im System (an verschiedenen Stationen) befinden, beendet wird, können sich selbst bei einem streng linear aufgebauten Modell leicht unterschiedliche Anzahlen an eingetroffenen Kunden an den verschiedenen Stationen ergeben.

Zwischenankunftszeiten

Eine Zwischenankunftszeit ist die Zeitspanne zwischen dem Eintreffen von zwei unmittelbar aufeinander folgenden Kunden an einer Station. Es werden hier Mittelwerte, Standardabweichungen, Varianzen, Variationskoeffizienten und Quantile ausgegeben.

Die mittlere Zwischenankunftszeit wird mit E[I] bezeichnet (und z.B. in Sekunden oder Minuten gemessen). Der Kehrwert von E[I] ist die Ankunftsrate. Sie wird mit lambda (λ) (in in z.B. 1/Sekunde oder 1/Minute gemessen). Kleine Werte von E[I] bzw. große Werte von lambda bedeuten, dass innerhalb kurzer Zeit viele Kunden eintreffen.
