Werden während der Simulation die Pfade der einzelnen Kunden durch das Modell aufgezeichnet, so werden auf dieser Seite alle von mindestens einem Kunden eingeschlagenen Pfade ausgewiesen und es wird angegeben, wie häufig der jeweilige Pfad jeweils absolut und relativ in Bezug auf alle Kunden eingeschlagen wurde.

Auf diese Weise kann vielfach nachvollzogen werden, ob sich die Kunden wie bei der Modellierung geplant durch das System bewegen.
