Auf dieser Seite werden die an Statistik-Stationen erhobenen Daten ausgegeben.

Bedeutung der Spalten in dieser Tabelle:

  • Benutzerdefinierte Statistik: In der Statistik-Station eingestellter Bezeichner für die zu erfassende Kenngröße auf die sich die jeweilige Zeile bezieht.
  • Mittelwert: Mittelwert der erfassten Kenngröße
  • Standardabweichung: Standardabweichung der erfassten Kenngröße
  • Var: Varianz der erfassten Kenngröße
  • CV: Variationskoeffizient der erfassten Kenngröße
  • Sk: Schiefe der erfassten Kenngröße
  • Kurt: Exzess (Wölbung) der erfassten Kenngröße
  • Minimum: Minimaler Wert der erfassten Kenngröße
  • Maximum: Maximaler Wert der erfassten Kenngröße
  • 10%-Quantil: 10%-Quantil der Werte der erfassten Kenngröße. 10% der erfassten Werte waren höchstens so groß wie in dieser Zelle angezeigt.
  • 25%-Quantil: 25%-Quantil der Werte der erfassten Kenngröße. 25% der erfassten Werte waren höchstens so groß wie in dieser Zelle angezeigt.
  • 50%-Quantil: 50%-Quantil der Werte der erfassten Kenngröße. 50% der erfassten Werte waren höchstens so groß wie in dieser Zelle angezeigt. Dieser Wert entspricht damit dem Median der erfassten Kenngröße.
  • 75%-Quantil: 75%-Quantil der Werte der erfassten Kenngröße. 75% der erfassten Werte waren höchstens so groß wie in dieser Zelle angezeigt.
  • 90%-Quantil: 90%-Quantil der Werte der erfassten Kenngröße. 90% der erfassten Werte waren höchstens so groß wie in dieser Zelle angezeigt.
