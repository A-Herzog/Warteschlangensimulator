In dieser Tabelle wird erfasst, wie viele Kunden während der Simulation an den betreffenden Stationen eingetroffen sind. In der Anteil-Spalte wird ausgegeben, wie hoch der Anteil an an dieser Station eingetroffenen Kunden in Bezug zur Gesamtanzahl an simulierten Kunden ist. Einzelne Anteilwerte können dabei größer als 100% sein; dies bedeutet, dass Kunden die betreffende Station mehrfach durchlaufen haben.
