Bedeutung der Spalten in dieser Tabelle:

  • Station oder Kundentyp: Modellstation bzw. Kundentyp auf die/den sich die jeweilige Zeile bezieht.
  • Anzahl: Anzahl Kundenereignissen an dieser Station während der gesamten Simulation.
  • E[.]: Mittlerer zeitlicher Abstand (in Sekunden) zwischen zwei entsprechenden Ereignissen an dieser Station.
  • Std[.]: Standardabweichung der zeitlichen Abstände (in Sekunden) zwischen zwei entsprechenden Ereignissen an dieser Station.
  • Var[.]: Varianz der zeitlichen Abstände (in Sekunden) zwischen zwei entsprechenden Ereignissen an dieser Station.
  • CV[.]: Variationskoeffizient der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station.
  • Sk[.]: Schiefe der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station.
  • Kurt[.]: Exzess (Wölbung) der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station.
  • Min[.]: Minimaler zeitlicher Abstand (in Sekunden) zwischen zwei entsprechenden Ereignissen an dieser Station.
  • Max[.]: Maximaler zeitlicher Abstand (in Sekunden) zwischen zwei entsprechenden Ereignissen an dieser Station.
  • 10%-Quantil[.]: 10%-Quantil der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station. 10% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
  • 25-Quantil[.]: 25%-Quantil der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station. 25% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
  • 50%-Quantil[.]: 50%-Quantil der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station. 50% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt. Dieser Wert entspricht damit dem Median der Zeitabstände.
  • 75%-Quantil[.]: 75%-Quantil der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station. 75% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
  • 90%-Quantil[.]: 90%-Quantil der zeitlichen Abstände zwischen zwei entsprechenden Ereignissen an dieser Station. 90% der zeitlichen Abstände waren höchsten so lang (in Sekunden) wie in dieser Zelle angezeigt.
