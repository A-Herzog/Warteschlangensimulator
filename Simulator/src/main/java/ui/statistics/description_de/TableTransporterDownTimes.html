Bedeutung der Spalten in dieser Tabelle:

  • Ressource: Transportergruppe auf die sich die jeweilige Zeile bezieht.
  • Anzahl an Transportern: Anzahl an Transportern in der jeweiligen Gruppe.
  • Anteil in Pause: Diese Zelle gibt den mittleren Anteil an Transportern in der Transportergruppe an, die sich in Pausezeit befinden.
  • Mittelwert: Gibt die mittlere absolute Anzahl an Transportern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Std: Standardabweichung der absoluten Anzahl an Transportern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Var: Varianz der absoluten Anzahl an Transportern in dieser Gruppe an, die sich in Pausezeit befinden.
  • CV: Variationskoeffizient der absoluten Anzahl an Transportern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Sk: Schiefe der absoluten Anzahl an Transportern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Kurt: Exzess (Wölbung) der absoluten Anzahl an Transportern in dieser Gruppe an, die sich in Pausezeit befinden.
  • Min: Minimale absolute Anzahl an während der Simulation gleichzeitig in Pausezeit befindliche Transporter in dieser Gruppe.
  • Max: Maximale absolute Anzahl an während der Simulation gleichzeitig in Pausezeit befindliche Transporter in dieser Gruppe.
