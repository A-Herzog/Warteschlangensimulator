Die Warte-, Transfer- und Bedienzeiten werden nicht nur für die Kunden auf ihren Wegen durch das System ausgewiesen, sondern auch die Stationen messen die an ihnen entstandenen Zeitdauern der verschiedenen Typen. Bei einem Modell mit nur einer Station entsprechen die Wartezeiten usw. über alle Kunden den Wartezeiten an der Station. Bei einem Modell mit mehreren Stationen entspricht die Summe der an allen Stationen entstandenen Wartezeiten der Summe der von allen Kunden aufgesammelten Wartezeiten. Innerhalb einer Station werden die Zeiten gesamt sowie nochmals untergliedert nach Kundentypen aufgezeichnet. Auf dieser Seite werden die Daten innerhalb einer Station aufgeschlüsselt nach Kundentypen ausgegeben.

Die mittleren Wartezeiten werden mit E[W] bezeichnet, die mittleren Transferzeiten mit E[T] und die mittleren die mittleren Bediendauern mit E[S].

Für alle Zeitangabe werden Mittelwerte, Standardabweichungen, Varianzen, Variationskoeffizienten, Schiefe, Exzess (Wölbung) und Quantile ausgegeben.

Der Flussgrad gibt das Verhältnis von Verweilzeit zu Bedienzeit ein. Der Flussgrad ist folglich immer größer oder gleich 1. Kleine Werte bedeuten, dass im Verhältnis zur Bedienzeit nur wenig Warte- und Transferzeiten angefallen sind. Große Werte deuten auf verhältnismäßig lange Warte- und Transferzeiten hin.

Konfidenzintervalle

Zu den auf dieser Seite angezeigten Größen können optional Konfidenzintervalle ausgegeben werden. Zur Berechnung von Konfidenzintervallen müssen die einzelnen Messwerte stochastisch unabhängig sein, was bei den in einem Warteschlangensystem erfassten Größen üblicherweise nicht der Fall ist. Um dennoch Konfidenzintervalle bestimmen zu können, wird daher eine feste Anzahl an Werten zu einem Batch zusammengefasst. Die Batch-Größen werden dabei so gewählt, dass zwischen diesen (fast) keine Abhängigkeiten mehr bestehen. (Diese virtuelle Zusammenfassung von Messwerten hat nichts mit Ankunfts- oder Bedien-Batchen zu tun.) Die Mittelwerte pro Batch werden dann als Eingangsgröße für die Berechnung der Konfidenzintervalle verwendet.

Die Erfassung von Batch-Means-basierenden Konfidenzintervallen kann auf der Dialogseite Ausgabeanalyse im Modelleigenschaften-Dialog aktiviert werden. Dafür muss eine Batch-Größe vorgegeben werden. Diese kann vorab über den Menüpunkt Statistik-Batch-Größe bestimmen im Simulation-Menü automatisch bestimmt werden.
