Werden in dem Modell Zähler-Stationen verwendet, so werde auf dieser Seite die Zählerstände am Ende der Simulation ausgegeben. Sind mehrere der Zähler zu einer Gruppe zusammengefasst, so wird außerdem der Anteil eines jeden Zählers in seiner Gruppe ausgewiesen. Dies kann z.B. verwendet werden, um zu erfassen, wie viele Kunden (gesamt und anteilig) jeweils bestimmte Pfade in dem Modell eingeschlagen haben.
