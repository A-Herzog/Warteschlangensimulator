Auf dieser Seite wird angezeigt, wie lange sich die Kunden im System aufgehalten haben. Die gesamte Zeitdauer von Ankunft bis Abgang aus dem System wird Verweilzeit genannt. Die mittlere Verweilzeit wird mit E[V] bezeichnet. Die Verweilzeit setzt sich additiv aus Warte-, Transfer- und Bedienzeit zusammen. Diese werden mit E[W], E[T] und E[S] bezeichnet.

Die Daten werden sowohl global, über alle Kundentypen als auch individuell pro Kundentyp ausgewiesen. (Existiert nur ein Kundentyp im System, so entfällt diese Unterteilung.)

Für alle Zeitangabe werden Mittelwerte, Standardabweichungen, Varianzen, Variationskoeffizienten und Quantile ausgegeben.

Der Flussfaktor gibt das Verhältnis von Verweilzeit zu Bedienzeit ein. Der Flussfaktor ist folglich immer größer oder gleich 1. Kleine Werte bedeuten, dass im Verhältnis zur Bedienzeit nur wenig Warte- und Transferzeiten angefallen sind. Große Werte deuten auf verhältnismäßig lange Warte- und Transferzeiten hin.
