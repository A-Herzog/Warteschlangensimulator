In dieser Tabelle wird eine Häufigkeitsverteilung einer oder mehrerer Zustandsgrößen ausgegeben. In der ersten Spalte jeder Zeile wird jeweils der Zustandswert angegeben, auf den sich die Zeile bezieht. In den weiteren Spalten wird angegeben, zu einem wie großen Anteil der Simulationszeit sich das System in dem jeweiligen Zustand befand.

Die in der letzten Zeile angezeigten relativen Häufigkeiten beziehen sich auf den in der ersten Spalte angezeigten Zustandswert und alle größeren Werte. D.h. die relativen Häufigkeiten in der letzten Zeile bilden die Summen über alle Zustandswerte die größer als die zuvor angegebenen Werte sind.
