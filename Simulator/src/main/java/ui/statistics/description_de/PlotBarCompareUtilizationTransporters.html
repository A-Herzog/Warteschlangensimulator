In diesem Diagramm wird die Auslastung der einzelnen Transportergruppen dargestellt. Jeder gestapelte Balken steht für eine Transportergruppe. Die Balken unterteilen sich in Bereiche für die mittlere Anzahl an fahrenden Transporter, an Transportern in Leerlaufzeit und an Transportern in Pausezeit. Auf der y-Achse sind jeweils die Mittelwerte als absolute Anzahlen an Transportern aufgetragen.
