Bedeutung der Spalten in dieser Tabelle:

  • Ressource: Bedienergruppe auf die sich die jeweilige Zeile bezieht.
  • Anzahl Bediener: Anzahl an Bedienern in der jeweiligen Gruppe, sofern diese durch eine feste Zahl (und nicht z.B. über einen variablen Schichtplan) festgelegt ist.
  • rho: Ist die Anzahl an Bedienern in dieser Gruppe ein während der Simulation fester Wert, so gibt rho die mittlere Auslastung der Bedienergruppe an.
  • Im Mittel belegt: Gibt die mittlere absolute Anzahl an ausgelasteten Bedienern in dieser Gruppe an.
  • Std: Standardabweichung der absoluten Anzahl an ausgelasteten Bedienern in dieser Gruppe an.
  • Var: Varianz der absoluten Anzahl an ausgelasteten Bedienern in dieser Gruppe an.
  • CV: Variationskoeffizient der absoluten Anzahl an ausgelasteten Bedienern in dieser Gruppe an.
  • Sk: Schiefe der absoluten Anzahl an ausgelasteten Bedienern in dieser Gruppe an.
  • Kurt: Exzess (Wölbung) der absoluten Anzahl an ausgelasteten Bedienern in dieser Gruppe an.
  • Min: Minimale absolute Anzahl an während der Simulation ausgelasteten Bedienern in dieser Gruppe an.
  • Max: Maximale absolute Anzahl an während der Simulation ausgelasteten Bedienern in dieser Gruppe an.
  • Leerlaufanteil (aller Bediener): Gibt den Zeitanteil aus, während der kein einziger Bediener der Gruppe arbeitete.
  • Volllastanteil: Ist die Anzahl an Bedienern in dieser Gruppe ein während der Simulation fester Wert, so gibt diese Zelle den Zeitanteil an, während der alle Bediener beschäftigt waren.

Die Größe rho gibt das Verhältnis von ausgelasteten und insgesamt verfügbaren Bedienern an. Ist die Anzahl an verfügbaren Bedienern in der Gruppe entweder variable (Definition über einen Schichtplan) oder aber besteht die Gruppe aus unendlich vielen Bedienern, so kann für rho kein Wert bestimmt werden. Dasselbe gilt für den Volllastanteil: Existiert keine feste Anzahl an vorhandenen Bedienern, so kann auch kein Zeitanteil in dem diese Anzahl beschäftigt war, ausgegeben werden.

Werte, die sich auf die absolute Anzahl an ausgelasteten Bedienern beziehen, stehen jedoch immer zur Verfügung.
