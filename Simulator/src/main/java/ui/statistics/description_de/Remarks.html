Auf dieser Seite werden Hinweise zu Kenngrößen, die außerhalb des normal üblichen liegen, aufgeführt. Diesen Kenngrößen sollte dann bei der Suche nach Flaschenhälsen oder Unstimmigkeiten im Modell ein besonderes Augenmerk gewidmet werden.

Auf dieser Seite können (je nach konkretem Wert der jeweiligen Größe) folgen Informationen ausgegeben werden:

Hinweise auf Fehler oder Warnungen

Wurde die Simulation aufgrund eines Fehlers abgebrochen oder sind während der Simulation Probleme aufgetreten, die zwar nicht unmittelbar zu einem Abbruch der Simulation führten, aber als Warnung erfasst wurden, so werden diese hier ausgegeben. Da Probleme während der Simulation häufig zu nicht verwendbaren bzw. nicht validen Ergebnissen führen, sollte die Ursache für diese Meldungen auf jeden Fall behoben werden.

Externe Datenquellen und Einschwingphasen

Werden Kundenankünfte aus einer externen Datenquelle geladen, so sollte keine Einschwingphase verwendet werden, da andernfalls einige der geladenen Kunden nicht in der Statistik erfasst werden. Werden in einem Modell externen Datenquellen und eine Einschwingphase kombiniert, so wird eine entsprechende Warnung ausgegeben.

Viele Kunden im System

Befinden sich sehr viele Kunden an einer Station oder im System, so deutet dies auf eine Überlastsituation an der betreffenden Station bzw. insgesamt hin. Außerdem wird die mittlere Anzahl an Kunden an einer Station zu dem Durchschnitt über die mittleren Anzahlen an Kunden an den anderen Stationen in Relation gesetzt. Befinden sich im Mittel an einer Station besonders viele Kunden im Vergleich zu den anderen Stationen, so ist auch dies ein Indiz für eine mögliche Fehlkonfiguration dieser Station.

Lange Wartezeiten

Sind die mittleren Wartezeiten an einer Station im Vergleich zu dem Durchschnitt über die mittleren Wartezeiten an den anderen Stationen sehr hoch, so kann dies bedeuten, dass diese Station einen Flaschenhals darstellt.

Hohe Flussgrad

Der Flussgrad ist der Quotient aus Verweilzeit zu Bedienzeit an einer Station. Ein Flussgrad von 1 bedeutet, dass keine Wartezeiten entstanden sind. Je höher der Flussgrad ist, desto mehr Wartezeiten sind (relativ zur Bedienzeit) aufgetreten. Hohe Flussgrade sind daher ein Indiz für ein schlecht dimensioniertes System. Es wird sowohl auf absolut gesehen hohe Flußssrade hingewiesen als auch darauf, wenn der Flussgrad an einer Station deutlich nach oben von dem Durchschnitt der Flussgrade über alle Stationen abweicht.

Hohe Variationskoeffizienten der Verweilzeiten

Hohe Variationskoeffizienten der Verweilzeiten bedeuten, dass die Durchlaufzeiten der Kunden durch das System nur sehr schlecht prognostiziert werden können. Daher wird bei realen Systemen meist versucht, hohe Variationskoeffizienten der Verweilzeiten zu vermeiden. Treten solche Werte in der Simulation insgesamt, an einer Station oder für einen Kundentyp auf, so gibt der Simulator eine entsprechende Warnung aus.

Hohe Auslastungen der Bediener

Zu hohe Auslastungen der Bediener führen in der Regel zu langen Wartezeiten der Kunden und sollten daher vermieden werden. Ist die Auslastung einer Bedienergruppe absolut gesehen besonders hoch oder ist die Auslastung einer Bedienergruppe im Verhältnis zu den Auslastungen der anderen Bedienergruppen besonders hoch, so wird eine entsprechende Warnung ausgegeben.
