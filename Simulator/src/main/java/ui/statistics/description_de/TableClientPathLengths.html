In dieser Tabelle werden die üblichen Kenngrößen (Mittelwert, Standardabweichung, Varianz, Variationskoeffizient, Schiefe, Exzess (Wölbung) und Quantile) der Pfadlängen der Pfade der Kunden durch das Modell angezeigt. Die Pfadlänge eines Kunden ist dabei die Anzahl an Stationen, die er in dem Modell durchlaufen hat. Die Quelle- und die Ausgangstation werden dabei auch als Stationen mitgezählt.
