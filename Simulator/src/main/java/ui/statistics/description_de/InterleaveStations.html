Eine Zwischenabgangszeit ist die Zeitspanne zwischen den Zeitpunkten, an denen zwei unmittelbar aufeinander folgende Kunden die Station verlassen haben. (Das "unmittelbar aufeinander folgen" bezieht sich dabei auf die Abgangszeitpunkte der Kunden, nicht auf die vorherigen Zeitpunkte der Ankünfte). Es werden hier Mittelwerte, Standardabweichungen, Varianzen, Variationskoeffizienten, Schiefe, Exzess (Wölbung) und Quantile ausgegeben.

Konfidenzintervalle

Zu den auf dieser Seite angezeigten Größen können optional Konfidenzintervalle ausgegeben werden. Zur Berechnung von Konfidenzintervallen müssen die einzelnen Messwerte stochastisch unabhängig sein, was bei den in einem Warteschlangensystem erfassten Größen üblicherweise nicht der Fall ist. Um dennoch Konfidenzintervalle bestimmen zu können, wird daher eine feste Anzahl an Werten zu einem Batch zusammengefasst. Die Batch-Größen werden dabei so gewählt, dass zwischen diesen (fast) keine Abhängigkeiten mehr bestehen. (Diese virtuelle Zusammenfassung von Messwerten hat nichts mit Ankunfts- oder Bedien-Batchen zu tun.) Die Mittelwerte pro Batch werden dann als Eingangsgröße für die Berechnung der Konfidenzintervalle verwendet.

Die Erfassung von Batch-Means-basierenden Konfidenzintervallen kann auf der Dialogseite Ausgabeanalyse im Modelleigenschaften-Dialog aktiviert werden. Dafür muss eine Batch-Größe vorgegeben werden. Diese kann vorab über den Menüpunkt Statistik-Batch-Größe bestimmen im Simulation-Menü automatisch bestimmt werden.
