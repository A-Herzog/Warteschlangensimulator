Erlang-B

Bei der Erlang-B-Formel wird davon ausgegangen, dass in dem Modell kein Warteraum existiert, so dass Kunden abgewiesen werden, wenn diese nicht unmittelbar bedient werden können.

Da die Erlang-B-Formel ursprünglich aus dem Telekommunikationssektor stammt, wird hier nicht von der Anzahl an Bedienern, sondern von der Anzahl an verfügbaren Leitungen N gesprochen. Die Größe a gibt die Arbeitslast - in diesem Kontext auch Verkehrsangebot genannt - an. Dieser Wert, der Quotient aus Ankunfts- und Bedienrate, gibt an, wie viele Bediener bzw. Leitungen auf jeden Fall notwendig sind, um die eintreffenden Kunden langfristig bedienen zu können.

Die Ausgabegröße P(B) gibt an, mit welcher Wahrscheinlichkeit ein neu eintreffender Kunde auf ein System stößt, in dem sich bereits N Kunden befinden (also alle Leitungen belegt sind). In diesem Fall wird der neue Kunde unmittelbar wieder abgewiesen. Bei der praktischen Dimensionierung eines Kommunikationssystems ist es das Ziel, dass P(B) möglichst klein ausfällt.
