Wartezeittoleranz

Die Wartezeittoleranz der Kunden in einem realen System lässt sich nicht direkt messen. Nur von den Warteabbrechern ist bekannt, wie lange diese bereit waren zu warten. Von den erfolgreichen Kunden ist nur eine untere Abschätzung der Wartezeittoleranz bekannt: Sie waren offenbar auf jeden Fall so lange bereit zu warten, wie dies notwendig war, bis sie bedient wurden. Die sinnvolle Schätzung der Wartezeittoleranz (die z.B. für die Verwendung der erweiterten Erlang-C-Formel notwendig ist) ist daher ein schwieriges Problem.

Eine mögliche Herangehensweise besteht darin, anzunehmen, dass die Wartezeittoleranzen exponentiell verteilt seien (was in der erweiterten Erlang-C-Formel sowieso ebenfalls angenommen wird). In diesem Fall lässt sich die mittlere Wartezeittoleranz E[WT] direkt als Quotient aus der mittleren Wartezeit der Kunden E[W] und der Abbruchrate P(A) bestimmen.
