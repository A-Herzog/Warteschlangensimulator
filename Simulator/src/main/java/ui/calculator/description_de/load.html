Auslastung

Die Auslastung ρ eines Bediensystems ist der Quotient aus der Ankunftsrate λ auf der einen Seite und dem Produkt aus Bedienrate μ und der Anzahl an Bedienern c auf der anderen Seite.

Damit ein Bediensystem langfristig stabil arbeiten kann, muss ρ<1 gelten, d.h. es müssen weniger Kunden eintreffen als die Bediener bedienen können.

Löst man die Formel ρ=λ/(c·μ) nach c auf, so ergibt sich: c=λ/(ρ·μ). Auf diese Weise kann bestimmt werden, wie viele Bediener nötig sind, um bei bekannter Ankunfts- und Bedienrate auf eine vorgegebene Auslastung zu kommen.

Grenzen der Rechnung

Bei dieser Rechnung wird nicht berücksichtigt, dass die konkreten Ankünfte zu zufälligen Zeitpunkten erfolgen und die konkreten Bedienzeiten unterschiedlich lange dauern. Daher liefert die obige Rechnung keine Informationen darüber, wie lange die Wartezeiten ausfallen, die sich in solch einem System ergeben würden.
