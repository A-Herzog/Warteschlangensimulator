The Pollaczek窶適hinchine formula allows the calculation of exact values for the characteristics in a queueing system with non-exponentially distributed service times. (The Erlang C formula assumes exponential distribution for both inter-arrival and service times. The Allen-Cunneen formula, on the other hand, which does not specify the distribution type for inter-arrival and service times, is only an approximation formula). For this, however, the Pollaczek窶適hinchine formula is only applicable for the case of a single operator system (c=1).

Performance indicators

The following parameters are calculated on the basis of the inputs:

  • Offered work load a / work load ρ: Since a single operator system is considered, we have a=ρ=λ/μ. The work load has always be <1, so that the operator can serve the arriving clients on a long-term average. However, a work load of less than 1 alone is not a statement that the system is sensibly dimensioned, since the work load does not provide any information about how long the clients will have to wait on average in the system.
  • Average number of clients in the system E[N]: This value indicates the average total number of clients in the system, i.e. it is the sum of waiting and currently served clients.
  • Average waiting time E[W]: Using Little's formula, the average queue length E [NQ] (and the arrival rate λ) can be used to calculate how long clients will have to wait on average before they are served.
  • Average residence time E[V]: The residence time is the sum of the average waiting time E[W] and the average service time E[S]. The average service time is the reciprocal of the service rate (E[S]=1/μ).
  • Flow factor E[V]/E[S]: The flow factor is the ratio of the residence time to the service time. Consequently, the flow factor is always greater than or equal to 1. Small values mean that in relation to the service time only little waiting time has occurred. Large values indicate a relatively long waiting time.
