Prepare table for table source

Many usual tables are structured in such a way that the first line contains the respective column headings and the following lines contain the respective values below the respective headings. Example of a table with client arrival times:

Time stamp Client type Costs Property
10Clients A5Default
17Clients B3Default
25Clients A11Special

However, for processing a table at a Table source, it has to look like this:

10Clients AClientData(1)=5ClientData("Property")=Default
17Clients BClientData(1)=3ClientData("Property")=Default
25Clients AClientData(1)=11ClientData("Property")=Special

The functionality offered in this dialog allows to translate tables from the form shown above to the form shown below. It is possible to choose which of the additional columns (in the example "Costs" and "Property") should be transferred and which client data index should be used for numeric columns (in the example "Costs" as "ClientData(1)").
