Data privacy settings

The data privacy settings dialog is accessible via the menu item of the same name in the help menu. The data privacy settings dialog offers collected privacy settings that are also accessible via the Settings dialog.

The settings offered in this dialog distinguish between two areas:

Local settings

This section contains settings related to the behavior of the program with respect to the handling of model files. It is possible to define which data are stored in model files and how model files should be handled in the simulator.

Internet

The settings in this section can be used to specify whether and in what way the simulator is allowed to access the Internet. Apart from actions explicitly triggered by the user (clicking links), the simulator only accesses the Internet for searching and downloading updates.
