Simulation results

On simulation results page, the results are displayed in the form of texts, tables and graphs after a simulation run. The data can in each case be exported in suitable formats (e.g., Excel tables or images).

Fast access

The fast access feature allows to present simulation results always in exactly the same user-defined format for example for saving them in table form. By using the filter settings at the bottom of this page you can define which data are to be exported and in which format.

The filtering can be done by list consisting of the records to be output and by scripts in Javascript or in Java. Depending on the selected language in the scripts some additional Javascript commands or additional Java commands are available for accessing the statistics data.

Generate report

The Generate report function allows to output a user-defined selection of statistics data as Word files, pdf files or as Excel workbooks.
