Batch processing

The batch processing function allows to apply two types of functions to each of the matching files in the selected directory:

Simulation

In this mode, all models and parameter series configurations located in the specified directory are processed. If a file is named Model.xml, the results for it are stored in Results-Model.zip. In this way, the simulator can also check for which input files simulation results are already available. If the processing of a directory is interrupted, only the models or parameter series configurations for which output files are not yet available are processed when processing is resumed. This functions corresponds to the command-line command Folder.

Filter results

In this mode, a specified filter script is applied to all statistics files present in a directory. The results are written collectively to a common output file specified by the user.

This functions corresponds to the command-line command FolderFilter.
