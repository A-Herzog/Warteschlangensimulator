Model element: Table

The table element allows to display data in table form during the animation of a model. Any computable expression can be displayed (which includes the functions for determining queue lengths at individual stations as well as access to all user-defined simulation variables). The output is in table form on the drawing surface. For classical simulation, the simulation data elements are without further meaning.

Settings

For the texts, you can set whether some special HTML and LaTeX symbols should be interpreted accordingly, whether formatting should be performed according to the Markdown symbols #, ##, ###, * and ** and whether the LaTeX symbols _{...}, ^{...}, \frac{...}{...} and \binom{...}{...} are to be interpreted.
