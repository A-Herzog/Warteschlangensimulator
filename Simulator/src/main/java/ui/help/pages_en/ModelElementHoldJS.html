Model element: Condition (script)

This element allows to hold and forward the incoming clients by using a Javascript-based or Java-based condition.

Settings

The script code can contain any Javascript or Java commands as well as some special Javascript commands or special Java commands for accessing the simulation system. If the condition contains values that can change independently of events (e.g. the simulated time), it may is necessary, to activate the option "Do additional time-based condition checks". In this case, the value of the condition is additionally checked in certain time intervals. How long these intervals are can be configured in the Model properties dialog. Additional time-based checks significantly slow down the simulation and should only be activated if this is for the condition necessary.
