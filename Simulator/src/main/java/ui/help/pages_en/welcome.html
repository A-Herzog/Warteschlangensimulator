Welcome to Wartenschlangensimulator

The Warteschlangensimulator allows to model, simulate and display animations of processes that can be interpreted as queueing systems.

First steps

  • As as first step you can load, view and simulate some example models. Click on "Load example" in the File menu.
  • A step by step turorial can be started by clicking on " Interactive tutorial" in the Help menu.
  • As short introduction about using the Warteschlangensimulators can be viewed via the menu item " Tutorial (pdf)" in the Help menu.
  • You can directly go to the Model editor and start a new model. Click on " Model editor" in the tool bar at the top of the window.
