Queueing calculator dialog

Mit Hilfe des Warteschlangenrechners können mit Hilfe analytischer Formeln Kenngrößen für einfache Warteschlangensysteme berechnet werden.

The queueing calculator allows to use analytical formulas to get performance indicators for some simple queueing models.

Work load

The work load ρ is calculated as the ratio of arrival rate λ and the service rate · number of agents μ · c. By term change the necessary number of agents can be determined to achieve a given work load for a given arrival and service rate.

Erlang B

The Erlang B formula gives the part of clients who will receives a busy signal when the traffic offered (=λ/μ) and the number of phone lines is known.

Erlang C

Using the Erlang C formula, you can calculate the part of clients who will have to wait longer than a certain number of seconds. For the Erlang C formula is assumed that both the interarrival times and the service times are exponentially distributed. Neither forwarding nor canceling clients nor retryers are considered.

Erlang C (extended)

The extended Erlang C formula differs from the ordinary Erlang C formula that it models cancel waiting clients. It is assumed that the wait time tolerances of the clients are exponentially distributed.

Allen-Cunneen

The Allen-Cunneen formula is an approximation formula for the parameters of a queuing system. In contrast to the Erlang formulas it does not require that interarrival and service times are exponentially distributed. Therefore here the coefficient of variation of these two parameters has to be set explicitly.

Wartezeittoleranz

Assuming that the waiting time tolerance is exponentially distributed (which usually is not the case), the average waiting time tolerance of the clients E[WT] can be calculated as the quotient of the mean waiting time E[W] and the cancelation rate P(A). This calculated value can be used as a first approximation for the waiting time tolerance calibration.
