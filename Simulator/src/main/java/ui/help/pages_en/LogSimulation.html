Record simulation to log file

The function for recording a simulation makes it possible to record all events processed during the simulation in a log file. In this way, the behavior of the simulator or the movement of individual clients through by the system can be precisely understood.

Since, depending on the complexity of the model and the selected file format, at least 1 KB of data per client is generated during the recording, it is recommended to limit the number of client to be simulated to a as small as possible value.

Note:
The program function "Log file analysis" allows to analyze a recorded simulation run graphically.
