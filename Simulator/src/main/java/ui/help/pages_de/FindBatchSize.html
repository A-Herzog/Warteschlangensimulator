Statistik-Batch-Größe bestimmen

Siehe auch Abschnitt Konfidenzintervalle bei Bediensystemen / Output-Analyse im Lehrbuch.

Diese Funktion simuliert das aktuelle Modell mit eingeschalteter Autokorrelations-Erfassung. Auf dieser Basis werden im Anschluss Vorschläge für die zu wählende Batch-Länge unterbreitet, so dass jeweils bestimmte Schwellenwerte in Bezug auf die Autokorrelation von Messpunkten, die nicht im selben Batch liegen, unterschritten werden. Die Betrachtung von Kenngrößen auf Batch-Basis ermöglicht es dann in späteren Simulationsläufen Konfidenzintervalle zu bestimmen.
