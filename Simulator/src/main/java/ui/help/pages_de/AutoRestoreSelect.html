Automatisch gespeichertes Modell zum Laden auswählen

Wenn die Option, dass das jeweils zuletzt verwendete Modell beim Programmende automatisch gespeichert werden soll, aktiviert ist und mehrere Programminstanzen verwendet wurden, kann es vorkommen, dass beim Programmstart mehrere gespeicherte Modelle zur Laden zur Auswahl stehen. In diesem Dialog kann ausgewählt werden, welches der Modelle geladen werden soll.
