Simulationsergebnisse verschiedener Modelle vergleichen Dialog

Die Vergleichsfunktion ermöglicht es, die Statistikergebnisse von bis zu 5 verschiedenen Simulationen parallel nebeneinander zu öffnen und anzuzeigen. Bei der Auswahl einer Rubrik in der Baumstruktur werden dann die jeweiligen Daten aller Modelle direkt nebeneinander angezeigt, um so einen Vergleich der Modelle zu ermöglichen.
