Hintergrund der Zeichenfläche Dialog

Der Hintergrund der Zeichenfläche Dialog kann über den Menüpunkt Ansicht|Hintergrundfarbe einstellen aufgerufen werden.

In diesem Dialog können die Hintergrundfarbe und die Rasterfarbe, die auf der Zeichenfläche verwendet werden sollen, eingestellt werden.

Hinweis:
Die Einstellung werden mit dem jeweiligen Modell gespeichert.
