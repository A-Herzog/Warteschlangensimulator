Datenschutzeinstellungen

Der Datenschutzeinstellungen-Dialog ist über den gleichnamigen Menüpunkt im Hilfe-Menü erreichbar. Im Datenschutzeinstellungen-Dialog werden gesammelt Einstellungen zum Datenschutz angeboten, die auch über den Einstellungen-Dialog zugänglich sind.

Bei den in diesem Dialog angebotenen Einstellungen wird zwischen zwei Bereichen unterschieden:

Lokale Einstellungen

In dieser Rubrik befinden sich Einstellungen, die sich auf das Verhalten des Programms in Bezug auf den Umgang mit Modelldateien beziehen. Es kann festgelegt werden, welche Daten in Modelldateien gespeichert werden und wie mit Modelldateien im Simulator umgegangen werden soll.

Internet

Über die Einstellungen in dieser Rubrik kann festgelegt werden, ob und in welcher Weise der Simulator auf das Internet zugreifen darf. Abgesehen von explizit vom Nutzer ausgelösten Aktionen (das Anklicken von Links) greift der Simulator nur für die Suche und den Download von Updates auf das Internet zu.

Insbesondere besitzt der Warteschlangensimulator keinerlei Telemetriefunktionen oder ähnliches. Automatisierte Zugriffe auf andere externe Seiten abgesehen von der GitHub-Update-Adresse finden nicht statt.
