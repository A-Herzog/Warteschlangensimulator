Kommandzeilenbefehl ausführen Dialog

Siehe auch Abschnitt Kommandozeilenbetrieb im Lehrbuch.

In diesem Dialog werden alle Befehle, die als Kommandozeilenparameter beim Aufruf des Warteschlangensimulators genutzt werden können, aufgelistet. Zusätzlich wird zu allen Befehlen die Beschreibung der Funktionsweise angezeigt. Die Befehle können auch direkt innerhalb dieses Dialogs ausgeführt werden.
