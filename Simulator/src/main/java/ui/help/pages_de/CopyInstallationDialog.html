Warteschlangensimulator-Installation übertragen Dialog

Vielfach ist es wünschenswert, den Warteschlangensimulator z.B. auf einem USB-Stick zu installieren, um so ein Modell während einer Präsentation auf einem entfernten Rechner vorführen zu können. Dies ist jederzeit durch einen Download der zip-Variante des Simulators von

github.com/A-Herzog/Warteschlangensimulator

und das anschließende Entpacken in dem Zielverzeichnis möglich. Allerdings kann auch eine normale Installation jederzeit - ohne erneuten Download - direkt in ein anderes Verzeichnis kopiert werden.

Die in diesem Dialog angebotene Funktion automatisiert diesen Prozess. Es muss dafür lediglich das gewünschte Zielverzeichnis angegeben werden.

Hinweis: Diese Programmfunktion wird nicht angeboten, wenn der Simulator bereits im portablen Modus läuft. In diesem Fall kann der Installationsordner einfach mit einem Dateimanager kopiert werden.
