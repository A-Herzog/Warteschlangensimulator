Imports für nutzerdefinierten Java-Code bearbeiten Dialog

Java-Code, der an den verschiedenen Stationen verwendet werden kann, wird automatisch mit einer Reihe von Import-Anweisungen ausgestattet, so dass innerhalb des Codes auf die üblichen Hilfsklassen zugegriffen werden kann. In diesem Dialog können die Vorgabewerte für die Imports verändert werden, um so ggf. weitere Klassen innerhalb des nutzerdefinierten Java-Codes zugänglich zu machen oder aber um die Nutzung der Hilfsklassen weiter einzuschränken.
