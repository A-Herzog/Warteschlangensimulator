Schiefe

Die Schiefe ist ein Maß für die Asymmetrie einer Messreihe. Eine Schiefe von 0 bedeutet, dass die Wahrscheinlichkeitsverteilung bzw. die Messreihe symmetrisch ist. Messreihen mit einer Schiefe kleiner als 0 werden als linksschief bezeichnet, Messreihen mit einer Schiefe größer als 0 als rechtsschief.

Formal handelt es sich bei der Schiefe um das auf die Standardabweichung normierte zentrale dritte Moment einer Wahrscheinlichkeitsverteilung.

Die symmetrische Normalverteilung besitzt eine Schiefe von 0. Die Exponentialverteilung besitzt eine Schiefe von 2, d.h. ist stets rechtsschief. Genauso sind die Lognormalverteilung und die Gamma-Verteilung stets rechtsschief. Die Beta-Verteilung kann links- oder rechtsschief sein.
