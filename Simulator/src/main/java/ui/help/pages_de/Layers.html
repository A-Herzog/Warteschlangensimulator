Ebenen-Dialog

Die Elemente auf der Zeichenfläche können verschiedenen Ebenen zugeordnet werden. Jedes Element kann dabei mehreren Ebenen zugeordnet werden; werden alle Zuordnungen aufgehoben, so gilt es als allen Ebenen zugehörig. Ein Element wird nur dann auf der Zeichenfläche angezeigt, wenn mindestens eine der Ebenen, auf denen es sich befindet, sichtbar ist.
