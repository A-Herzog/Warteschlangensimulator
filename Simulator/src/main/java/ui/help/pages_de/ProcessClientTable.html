Tabelle für Tabellenquelle aufbereiten

Siehe auch Abschnitt Laden von Werten während der Simulation im Lehrbuch.

Viele normale Tabellen sind so aufgebaut, dass in der erste Zeile die jeweiligen Spaltenüberschriften stehen und in den folgenden Zeilen unterhalb der jeweiligen Überschriften die jeweiligen Werte. Beispiel für eine Tabelle mit Kundenankunftszeiten:

Zeitpunkt Kundentyp Kosten Eigenschaft
10Kunden A5Normal
17Kunden B3Normal
25Kunden A11Speziell

Soll eine Tabelle ohne zusätzliche Konfiguration an einer Tabellenquelle verwendet werden, so muss die Tabelle jedoch wie folgt aussehen:

10Kunden AClientData(1)=5ClientData("Eigenschaft")=Normal
17Kunden BClientData(1)=3ClientData("Eigenschaft")=Normal
25Kunden AClientData(1)=11ClientData("Eigenschaft")=Speziell

Die in diesem Dialog angebotene Funktionalität erlaubt es, Tabellen aus der oben dargestellten Form in die unten dargestellte Form zu übersetzen. Es kann dabei gewählt werden, welche der zusätzlichen Spalten (im Beispiel "Kosten" und "Eigenschaft") übertragen werden sollen und welche Kundendaten-Index numerische Spalten (im Beispiel "Kosten" als "ClientData(1)") erhalten sollen.
