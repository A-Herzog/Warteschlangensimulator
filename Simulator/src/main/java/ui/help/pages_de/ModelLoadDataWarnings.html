Fehler beim Laden der externen Daten

Über den Menüpunkt "Externe Modelldaten" im Modell-Menü können externe Daten (in Form von Tabellenzellen) definiert werden, die entweder manuell oder automatisch unmittelbar vor dem Start einer Simulation als Parameter in das Modell geladen werden. Treten bei diesem Ladevorgang Fehler auf, entweder weil die Quelle nicht verfügbar ist oder weil die geladenen Werte nicht für die jeweiligen Parameter verwendet werden können, so werden diese in diesem Dialog beschrieben.
