Modell-Element: Mehrfach-Zuweisung

Das Mehrfach-Zuweisung-Element weist den Kunden, die es passieren, einen neuen Kundentyp dem sie in der Statistik zugeordnet werden sollen, zu. Es können dabei verschiedene Kundentypen in Abhängigkeit von konfigurierbaren Bedingungen zugewiesen werden.

Einstellungen

Es können beliebig viele Kundentypen angegeben werden und für jeden eine Bedingung, wann diese Zuweisung angewendet werden soll.

Die optionale Bedingung ermöglicht es, dass nur unter bestimmten Voraussetzungen eine Zuweisung erfolgt, wenn ein Kunde die Station passiert.
