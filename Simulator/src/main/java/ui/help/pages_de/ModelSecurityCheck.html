Sicherheitsprüfung

Soll ein Modell geladen werden, das Elemente enthält, die evtl. Auswirkungen auf das System haben können (Stationen, die Skriptausführung beinhalten oder die Daten in Dateien oder Datenbanken ausgeben), so wird, wenn dies in den Einstellungen aktiviert ist, ein Warndialog angezeigt, in dem dargestellt ist, welche Stationen mit welchen besonderen Eigenschaften das Modell enthält. Es kann kann entschieden werden, ob das Modell geladen werden soll oder ob der Ladevorgang abgebrochen werden soll.
