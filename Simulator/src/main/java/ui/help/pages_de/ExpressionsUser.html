Benutzerdefinierte Funktionen Dialog

In diesem Dialog können zusätzliche Funktionen definiert werden, die dann in allen Rechenausdrücken als reguläre Funktionen zur Verfügung stehen. Innerhalb einer nutzerdefinierten Funktion stehen alle normalen Rechenausdrücke zur Verfügung. Bei der Definition einer nutzerdefinierten Funktion muss angegeben werden, wie viele Parameter diese besitzen sollen. Die Parameter werden wie bei einer eingebauten Funktion beim Aufruf nach dem Funktionsnamen in runden Klammern übergeben. Innerhalb der nutzerdefinierten Funktion stehen die übergebenen Werte in den Variablen "Parameter1", "Parameter2" usw. zur Verfügung.
