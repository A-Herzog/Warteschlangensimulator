Vorlage für neue Modelle

Beim Aufruf des Menüpunktes "Neu" im "Datei"-Menü wird im Normalfall ein neues, leeres Modell mit intern vorgegebenen Initialeinstellungen erstellt. Es kann jedoch auch eine externe Datei hinterlegt werden, von der ein neues Modell beim Aufruf der Menüfunktion abgeleitet wird. Diese Datei muss den Namen Normal.xml besitzen und sich entweder im Einstellungen-Verzeichnis oder im Programm-Verzeichnis des Simulators befinden.
