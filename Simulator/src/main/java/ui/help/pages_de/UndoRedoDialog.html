Rückgängig/Wiederholen-Schritt auswählen Dialog

In diesem Dialog werden alle Modellzustände, die im Rückgängig-Wiederholen-Speicher vorhanden sind, angezeigt. Über die "Ok"-Schaltfläche kann zu dem jeweils gewählten Zustand au der Zeichenfläche zurückgekehrt werden. Über "Speichern" kann einer der Zwischenzustände als Datei gespeichert werden, ohne dass dadurch das Modell im Editor verändert wird.
