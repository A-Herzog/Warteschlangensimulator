Verteilung anpassen Dialog

Siehe auch Abschnitt Verteilungsanpassung im Lehrbuch.

Die Verteilungsanpassung kann auf zwei Arten erfolgen: Es kann entweder eine einzelne Verteilungsfunktion (aus einer großen Menge möglicher Verteilungsfunktionen) gesucht werden, die die angegebenen Messwerte möglichst gut beschreibt. Oder aber die Messwerte können als eine Überlagerung von mehreren Verteilungen (mit vorgegebenem, bekanntem Typ) angenommen werden. Die Verteilungsanpassung versucht dann aus dieser multimodalen Messreihe die einzelnen Verteilungen zu extrahieren.

Ermittlung einer einzelnen, die Messwerte möglichst gut beschreibenden Verteilungsfunktion

Die Funktion zur Verteilungsanpassung ermöglicht es, basierend auf einer Reihe von Messwerten eine Verteilung zu finden, die die Werte jeweils bestmöglich widerspiegelt. Es werden dafür zunächst Mittelwert und Standardabweichung der Messwerte bestimmt und basieren darauf für alle in das Programm eingebaute Wahrscheinlichkeitsverteilungen entsprechende Verteilungsfunktionen aufgestellt. Zum Abschluss werden alle diese konkreten Verteilungsfunktionen mit der empirischen Verteilung der Messwerte verglichen und es wird jeweils die Abweichung bestimmt. Die Verteilungsfunktion, die die geringste Abweichung zu den Messwerten aufweist, wird als angepasste Verteilungsfunktion empfohlen.

Zusätzlich werden bei allen Verteilungen p-Werte für verschiedene Anpassungstests ausgegeben. Je höher ein p-Wert ist, desto schwerer ist es, die Hypothese abzulehnen, dass der Messreihe die jeweilige Verteilung zugrunde liegt. Für alle Verteilungen werden der Kolmogorov-Smirnov-Anpassungstest und der Chi2-Anpassungstest durchgeführt. Während der Kolmogorov-Smirnov-Anpassungstest insbesondere die maximalen Abweichungen zwischen Messreihe und Verteilungsfunktion bewertet, betrachtet der Chi2-Anpassungstest die mittleren Abweichungen. Für die Normalverteilung wird zusätzlich der Anderson-Darling-Test, der eine Modifikation des Kolmogorov-Smirnov-Tests darstellt, durchgeführt.

Zerlegung einer Messreihe in mehrere Verteilungen

In dieser Betriebsart muss zunächst ausgewählt werden, ob Log-Normal- oder Gamma-Verteilungen für die einzelnen Verteilungen herangezogen werden. Die Verteilungsanpassung versucht dann sukzessive Verteilungen zu finden, die jeweils die Messreihe insgesamt möglichst gut beschreiben. Solch eine Teilverteilung wird dann von der vorgegebenen Gesamtverteilung abgezogen und der Algorithmus wird erneut auf die verbleibende Verteilung angewandt. Dieser Prozess wird iteriert, bis die Summe der Teilverteilungen mindestens 99% der Messwerte-Verteilung beschreibt.
