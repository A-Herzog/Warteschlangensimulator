Benutzerdefinierte Animationsicons Dialog

Der Benutzerdefinierte Animationsicons Dialog kann über den Menüpunkt Ansicht|Benutzerdefinierte Animationsicons aufgerufen werden.

In diesem Dialog können Icons geladen werden, die dann im Folgenden für die Animation von Kunden, Bedienern und Transportern zur Verfügung stehen.

Hinweis:
Die Einstellung werden mit dem jeweiligen Modell gespeichert.
