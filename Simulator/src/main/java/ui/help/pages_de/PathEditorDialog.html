Wegstrecken bearbeiten Dialog

Siehe auch Abschnitt Transporte mit Transportern im Lehrbuch.

Bei jedem Wegpunkt kann eingestellt werden, in welcher Reihenfolge dieser von Transportern auf dem Weg von welchem Startpunkt zu welchem Zielpunkt angesteuert werden soll. Bei größeren Modellen kann diese Konfiguration schnell aufwendig werden. Der Wegstrecken bearbeiten Dialog hilft, diesen Prozess zu vereinfachen. In diesem Dialog können für jede Station eine oder mehrere mögliche Folgestationen definiert werden. Wird der Dialog mit "Ok" geschlossen, so versucht der Simulator auf dieser Basis Wege von jeder Station zu jeder anderen Station zu finden und konfiguriert die Wegpunkte, so dass diese jeweils in der richtigen Reihenfolge auf dem jeweils richtigen Weg angesteuert werden.
