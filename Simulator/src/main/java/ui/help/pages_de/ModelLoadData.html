Externe Modelldaten bearbeiten Dialog

Über diesen Dialog können bestimmte Zellen in einer externen Tabelle definiert werden, deren Werte als Parameter in das Modell geladen werden sollen. Der Ladevorgang kann manuell ausgelöst werden oder aber automatisch vor dem Start einer Simulation. Auf diese Weise kann ein Modell dynamisch mit externen Daten parametrisiert werden.
