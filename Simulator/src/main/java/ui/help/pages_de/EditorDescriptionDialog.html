Modellbeschreibung-Dialog

Der Modellbeschreibung-Dialog kann über den Menüpunkt Modell|Modellbeschreibung aufgerufen werden.

In diesem Dialog werden die Werte der wesentlichen Parameter der Stationen, die für den Simulationsablauf von Bedeutung sind, in Form einer Modellbeschreibung angezeigt. Die Modellbeschreibung kann dabei in verschiedenen Formaten exportiert werden.
