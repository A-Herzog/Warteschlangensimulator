Simulationslauf aufzeichnen

Siehe auch Abschnitt Aufzeichnung von Simulationsläufen im Lehrbuch.

Die Funktion zur Aufzeichnung einer Simulation ermöglicht es, alle während der Simulation abgearbeiteten Ereignisse in einer Datei festzuhalten. Auf diese Weise kann das Verhalten des Simulators bzw. die Bewegung einzelner Kunden durch das System genau nachvollzogen werden.

Da bei der Aufzeichnung in Abhängigkeit von der Komplexität des Modells und des gewählten Dateiformats mindestens 1 KB an Daten pro Kunde entsteht, empfiehlt es sich, die Anzahl an zu simulierenden Kunden für die Aufzeichnung auf einen so kleinen wie möglichen Wert zu begrenzen.
