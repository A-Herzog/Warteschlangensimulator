Pfade visualisieren Dialog

Wurden die Kundenpfade aufgezeichnet (siehe Modelleigenschaften-Dialog) so kann über die entsprechenden Statistikseiten der Dialog zur Visualisierung der Kundenpfade aufgerufen werden. In diesem Dialog werden die gemäß ihrer Häufigkeit sortierten Pfade der Kunden durch das System direkt im Modell visualisiert.
