Warteschlangenrechner Dialog

Mit Hilfe des Warteschlangenrechners können mit Hilfe analytischer Formeln Kenngrößen für einfache Warteschlangensysteme berechnet werden.

Auslastung

Die Auslastung ρ berechnet sich als Quotient aus Ankunftsrate λ und Bedienrate·Agentenanzahl μ·c. Durch Termumstellung lässt sich so die notwendige Agentenanzahl bestimmen, um bei einer vorgegebenen Ankunfts- und Bedienrate eine bestimmte Auslastung zu erreichen.

Erlang B Formel

Die Erlang B Formel gibt an, ein wie großer Anteil an Kunden bei bekanntem Verkehrsangebot (=λ/μ) und bekannter Anzahl an Telefonleitungen ein Besetztzeichen erhält.

Erlang C Formel

Mit Hilfe der Erlang C Formel lässt sich berechnen, ein wie großer Anteil der Kunden höchstens eine gewisse Anzahl an Sekunden warten muss. Für die Erlang C Formel wird angenommen, dass sowohl die Zwischenankunftszeiten als auch die Bedienzeiten exponentiell verteilt sind. Auch werden weder Weiterleitungen noch Warteabbrecher noch Wiederholer berücksichtigt.

Erweiterte Erlang C Formel

Die erweiterte Erlang C Formel unterscheidet sich darin von der gewöhnlichen Erlang C Formel, dass sie Warteabbrecher modelliert. Dabei wird angenommen, dass die Wartezeittoleranzen der Kunden exponentiell verteilt sind.

Allen-Cunneen

Die Allen-Cunneen Formel ist eine Näherungsformel für die Kenngrößen eines Warteschlangensystems. Im Gegensatz zu den Erlang-Formeln setzt sie nicht voraus, dass Zwischenankunfts- und Bedienzeiten exponentiell verteilt sind. Dafür kann hier der Variationskoeffizient dieser beiden Parameter explizit eingestellt werden.

Wartezeittoleranz

Nimmt man an, dass die Wartezeittoleranz exponentiell verteilt ist (was üblicher Weise nicht der Fall ist), so lässt sich die mittlere Wartezeittoleranz der Kunden E[WT] berechnen als Quotient der mittleren Wartezeit E[W] und der Abbruchrate P(A). Der so berechnete Wert kann als erste Näherung bei der Wartezeittoleranz-Kalibrierung verwendet werden.
