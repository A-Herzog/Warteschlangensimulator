Element über ID auswählen Dialog

Dieser Dialog erlaubt es, einzelne Elemente des Simulationsmodells über ihre ID ausfindig zu machen. Wird der Dialog über die "Ok"-Schaltfläche geschlossen oder doppelt auf einen Eintrag in der Liste der Element geklickt, so wird das jeweilige Element auf der Modelloberfläche zentriert dargestellt und ausgewählt.
