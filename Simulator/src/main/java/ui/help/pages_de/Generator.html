Modellgenerator Dialog

Siehe auch Abschnitt Modellgenerator im Lehrbuch.

Dieser Dialog erlaubt es, einige einfache Modelle, die insbesondere für bestimmte Forschungsfragen von Interesse sind, mit wenigen Mausklicks zu erstellen. Es müssen lediglich die wesentlichen Eigenschaften des Warteschlangenmodells eingestellt werden, die konkrete Erstellung und Verknüpfung der Stationen sowie die weitere Konfiguration der Modelleigenschaften erfolgt dann automatisch.
