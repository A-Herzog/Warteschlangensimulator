Java-Plugins-Verzeichnis Dialog

In den Skript-Stationen kann von dem internen Java-Code aus Java-Code in externen Klassendateien aufgerufen werden. Dafür steht der Befehl sim.getSystem().runPlugin("className","methodName",data) zur Verfügung (siehe auch Dokumentation der Java-Befehle). In diesem Dialog kann das Verzeichnis gewählt werden, in dem die externen Klassendateien gesucht werden sollen. In dem Ausgabebereich in der Mitte des Dialogs werden die jeweils gefundenen Klassendateien und die darin enthaltenen, passenden Methoden aufgelistet.
