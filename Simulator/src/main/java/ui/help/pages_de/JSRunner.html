Skript ausführen

Siehe auch Abschnitt Automatisierung von Simulationen mit Skripten im Lehrbuch.

Die Funktion zur Ausführung von Skripten ermöglicht es, Parameter in einem Simulationsmodell gezielt auf verschiedene bestimmte Werte zu setzen. Für jede der möglichen Parametervarianten kann dann eine Simulation durchgeführt werden. Die Ergebnisse können in Form einer Statistikdatei dokumentiert oder/und gefiltert ausgegeben werden. Für den Zugriff auf Modell und Statistikdaten stehen dabei je nach gewählter Sprache spezielle Javascript-Befehle bzw. spezielle Java-Befehle zur Verfügung.
