Verbindungskanten konfigurieren Dialog

Der Verbindungskanten konfigurieren Dialog kann über den Menüpunkt Ansicht|Verbindungskanten konfigurieren aufgerufen werden.

In diesem Dialog kann eingestellt werden, wie die Verbindungskanten zwischen den einzelnen Stationen dargestellt werden sollen.

Hinweis:
Die Einstellung werden mit dem jeweiligen Modell gespeichert.
