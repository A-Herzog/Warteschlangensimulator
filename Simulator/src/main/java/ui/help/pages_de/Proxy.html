Dialog: Proxy-Konfiguration

Dieser Dialog kann über den Einstellungen-Dialog aufgerufen werden.

Ausschließlich zur Suche nach Programmupdates greift der Warteschlangensimulator auf das Internet zu. Die Kommunikation erfolgt dabei ausschließlich mit dem Server qs.highfives.io und ausschließlich per https gesichert. Wenn keine direkte Internet-Verbindung möglich ist, sondern diese nur über einen Proxy möglich ist, können in diesem Dialog die Verbindungsdaten zu diesem Proxy-Server, der die Verbindung ins Internet vermittelt, eingetragen werden.
