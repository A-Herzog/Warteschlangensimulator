Parameterreihe simulieren

Siehe auch Abschnitt Parameterreihen im Lehrbuch.

Die Parameterreihe-Funktion ermöglicht es, einen oder mehrere Parameter in einem Simulationsmodell gezielt in einem bestimmten Bereich zu variieren. Für jede der möglichen Parametervarianten wird eine Simulation durchgeführt. Für jede Variante können dabei bestimmte Teilergebnisse direkt in der Ergebnistabelle dargestellt, als Statistikdatei gespeichert werden oder über ein Schnellzugriff-Javascript-Skript oder ein Schnellzugriff-Java-Skript gefiltert werden. Auf diese Weise lässt sich die Veränderung der relevanten Ergebnisse in Abhängigkeit von der Veränderung des Modells dokumentieren.
