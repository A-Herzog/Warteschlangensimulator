Simulations-Logdatei analysieren

Die Funktion zur Analyse von Simulations-Logdateien ermöglicht die Umwandlung eine textbasierten Logdatei in eine grafische Repräsentation.

Voraussetzung für die Analyse ist, dass bei der Aufzeichnung als Ausgabeformat eine Textdatei gewählt wurde und pro Ereignis eine Zeile ausgegeben wurde (keine Aufteilung von Ereignisname und Ereignisbeschreibung und keine Gruppierung von Ereignissen.)
