Heatmap konfigurieren Dialog

Nach dem Abschluss einer Simulation können die stationsbezogenen Kenngrößen in Form einer Heatmap direkt auf der Zeichenfläche visualisiert werden. In diesem Dialog kann das generelle Aussehen der Heatmap, d.h. die verwendeten Farben und die Größe der farbigen Bereiche um die Stationssymbole herum eingestellt werden.
