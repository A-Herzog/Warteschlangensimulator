Mittelwerte

Der Mittelwert ist die am häufigsten verwendete statistische Kenngröße. Er gibt den Durchschnitt der betrachteten Werte an und wird gebildet, in dem alle Messwerte aufsummiert werden und durch die Anzahl der Werte dividiert wird.

Beispiel

Es wurden die folgenden 4 Werte gemessen: 3, 5, 7, 5. Die Summe dieser Wert beträgt 20. Damit ergibt sich der Mittelwert zu (3+5+7+5)/4=5.

Bezeichnung

In der Stochastik wird formal unterschieden zwischen dem Mittelwert einer Messreihe und dem Erwartungswert der dem Experiment zugrunde liegenden Wahrscheinlichkeitsverteilung. Der Erwartungswert von X wird mit E[X] bezeichnet. Da in der Simulation Messreihen und Verteilungen sehr eng verwandt behandelt werden, wird auch für den Mittelwert die E[X]-Notation verwendet. Bezeichnet N die Anzahl an Kunden im System, so wird die mittlere Anzahl an Kunden im System mit E[N] bezeichnet. In der mathematischen Literatur werden Mittelwerte üblicherweise durch einen Balken über dem zugehörigen Bezeichner dargestellt.

Kritik

  • Die Verwendung von statistischen Kenngrößen bedeutet, dass eine große Anzahl an Messwerten auf eine einzige Größe komprimiert werden. Bei dieser Kompression gehen unvermeidlich Informationen verloren. So liefert der Mittelwert keine Informationen darüber, wie die Werte verteilt sind. So besitzen die Messreihen 0, 10, 0, 10, 0, 10 und 5, 5, 5, 5, 5, 5 beide denselben Mittelwert 5, obwohl diese vollständig unterschiedlich sind. Um auch die Verteilung der Werte abzubilden, wird daher häufig die Standardabweichung verwendet.
  • Der Mittelwert ist anfällig für einzelne Ausreißer. Die Messreihe 5, 5, 5, 5, 100 besitzt den Mittelwert 20, auch wenn alle Messungen bis auf einen den Wert 5 geliefert haben. Der Mittelwert entspricht also nicht "dem Wert in der Mitte". Ist nach diesem Wert in der Mitte der Messreihe gefragt, so muss das 50%-Quantil bzw. gleichbedeutend damit der Median betrachtet werden.
