Elemente ausrichten Dialog

Die in diesem Dialog angebotene Funktionen ermöglichen es, die Stationen auf der Zeichenfläche anzuordnen. Berücksichtigt werden dabei nur Stationen des Fließbildes. Elemente zur optischen Gestaltung werden nicht verschoben. Es kann ausgewählt werden, ob die Stationen lediglich an dem Grobraster ausgerichtet werden sollen oder ob das Fließbild basierend auf den Verknüpfungen zwischen den Stationen vollständig neu aufgebaut werden soll.
