Optimierer

Im Optimierer können für eine Reihe von Modellparametern Bereiche definiert werden, in denen diese variiert werden dürfen. Des Weiteren kann eine Zielgröße angegeben werden und eingestellt werden, ob diese maximiert, minimiert oder einen bestimmten Bereich einhalten soll. Der Optimierer versucht dann die vorgegebenen Modellparameter im Rahmen der jeweils zulässigen Bereiche so zu variieren, dass das gewünschte Optimum erreicht wird.

Das Optimierer-Fenster beinhaltet drei Seiten:

Kontrollvariablen

Auf dieser Seite kann eingestellt werden, welche Variablen in dem Modell in welchem Bereich variiert werden dürfen, um das gewünschte Ziel zu erreichen. Zusätzlich kann pro Variable angegeben werden, ob diese beliebige Zahlenwerte (in dem vorgegebenen Bereich) oder nur Ganzzahlen annehmen kann.

Ziel

Auf dieser Seite wird das Optimierungsziel vorgegeben. Es kann als Zielgröße entweder der Wert eines XML-Statistik-Elements verwendet werden oder aber der Rückgabewert eines Skripts. Des weiteren kann eingestellt werden, ob der Zielwert möglichst groß oder möglichst klein werden soll oder aber in einem bestimmten Zielbereich liegen soll.

Optimierung

Der Hauptbereich der Optimierung-Seite dient der Anzeige der Statusausgaben während des Optimierungsprozesses. Über diesem Bereich kann eingestellt werden in welchem Verzeichnis die Statistikdateien, die die einzelnen Optimierungsschritte repräsentieren, gespeichert werden sollen und ob während der Optimierung immer zunächst eine Variable auf ihren bestmöglichen Wert verändert werden soll und dann zur nächsten Variable gewechselt werden soll oder ob reihum alle Variablen jeweils ein Stück verändert werden sollen.
