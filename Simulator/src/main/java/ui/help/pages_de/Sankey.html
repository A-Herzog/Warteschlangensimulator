Kundenbewegungen als Sankey-Diagramm exportieren Dialog

Wurden die Stationsübergänge gezählt oder die Kundenpfade aufgezeichnet (siehe Modelleigenschaften-Dialog) so kann über die entsprechenden Statistikseiten der Dialog zur Erstellung eines Sankey-Diagramms aufgerufen werden. In dem Sankey-Diagramm werden die Pfade der Kunden durch das System durch verschieden breite Verbindungen der Stationen untereinander dargestellt. Die Aufzeichnung der Kundenpfade enthält mehr Informationen und liefert daher detailreichere Darstellungen.
