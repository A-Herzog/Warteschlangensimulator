Simulationsergebnisse

Auf Simulationsergebnisse Programmseite werden nach einem Simulationslauf die Ergebnisse in Form von Texten, Tabellen und Grafiken angezeigt. Die Daten können dabei jeweils in passenden Formaten (z.B. Excel-Tabellen oder Bilder) exportiert werden.

Schnellzugriff

Der Ergebnis-Schnellzugriff ermöglicht es, Daten von ganzen Simulationsreihen immer wieder in derselben Form z.B. zwecks Speicherung in Tabellen zu exportieren. Über die Filtereinstellungen am unteren Rand der Seite kann dabei sowohl eingestellt werden, welche Daten exportiert werden sollen, als auch wie die Ausgabe formatiert werden soll.

Die Filterung kann dabei sowohl in Form von Listen mit Einträgen als auch über Skripte in Javascript oder Java erfolgen. In den Skripten stehen in Abhängigkeit von der gewählten Sprache eine Reihe von zusätzlichen Javascript-Befehlen bzw. von zusätzlichen Java-Befehlen für den Zugriff auf die Statistikdaten zur Verfügung.

Zusammenfassung erstellen

Die Funktion Zusammenfassung erstellen ermöglicht es, eine benutzerdefinierte Auswahl der Statistikdaten als Word-Datei, als pdf-Datei oder aber als Excel-Arbeitsmappe zu speichern.
