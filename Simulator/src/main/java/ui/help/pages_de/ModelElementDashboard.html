Diagramm-Dashboard

Das Diagramm-Dashboard ist eine spezielle Zeichenfläche, auf der nur Diagramme und ähnliche Elemente, die nicht direkt in die Modell-Fließbilder eingebunden sind, eingefügt werden können. Diese Zeichenfläche kann während einer laufenden Animation parallel geöffnet sein, ohne dabei die Verwendung der Hauptzeichenfläche zu blockieren.
