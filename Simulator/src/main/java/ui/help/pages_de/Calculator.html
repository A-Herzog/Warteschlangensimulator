Rechner

Der Rechner erlaubt die Berechnung mathematischer Ausdrücke sowie die Anzeige von Funktionsverläufen auf Basis desselben Systems, das auch während der Simulation zur Berechnung von Ausdrücken verwendet wird.

Abzüglich der modellabhängigen Symbole, die nur während der Simulation zur Verfügung stehen, können alle üblichen mathematischen Symbole verwendet werden.
