Vergleich mit analytischem Modell-Dialog

Der Vergleich mit analytischem Modell-Dialog kann über den Menüpunkt Modell|Vergleich mit analytischem Modell aufgerufen werden.

In diesem Dialog werden Vergleichsdaten gemäß dem Erlang-C- und dem Allen-Cunneen-Modell ausgegeben. Liegen Simulationsergebnisse vor, so werden auch diese zum Vergleich von analytischen Modellen und Simulationsergebnissen, dargestellt. Damit diese Funktion verwendet werden kann, muss das Modell einfach genug aufgebaut sein, so dass das Programm eine einzige, eindeutige Quelle und eine einzige, eindeutige Bedienstation identifizieren kann.
