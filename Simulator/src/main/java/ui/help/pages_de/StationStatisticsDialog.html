Stationen in der Statistik erfassen Dialog

Im Normalfall werden die Anzahlen an Kunden an den Stationen (gesamt und wartend) sowie die Zwischenankunfts- und die Zwischenabgangszeiten an allen Stationen erfasst. In diesem Dialog kann eingestellt werden, dass für bestimmte Stationen keine Erfassung dieser Daten erfolgen soll. Dies kann aus Übersichtlichkeitsgründen in der Statistik wünschenswert sein. Außerdem erhöht eine reduzierte Erfassung von Statistikdaten die Simulationsgeschwindigkeit geringfügig.

Solange es keine unmittelbaren Gründe gibt, die Statistikerfassung zu reduzieren, wird empfohlen, stets die Erfassung der Daten an allen Station aktiv zu lassen.
