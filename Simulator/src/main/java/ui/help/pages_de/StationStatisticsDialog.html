Stationen in der Statistik erfassen Dialog

Statistikerfassung

Im Normalfall werden die Anzahlen an Kunden an den Stationen (gesamt und wartend) sowie die Zwischenankunfts- und die Zwischenabgangszeiten an allen Stationen erfasst. In diesem Dialog kann eingestellt werden, dass für bestimmte Stationen keine Erfassung dieser Daten erfolgen soll. Dies kann aus Übersichtlichkeitsgründen in der Statistik wünschenswert sein. Außerdem erhöht eine reduzierte Erfassung von Statistikdaten die Simulationsgeschwindigkeit geringfügig.

Solange es keine unmittelbaren Gründe gibt, die Statistikerfassung zu reduzieren, wird empfohlen, stets die Erfassung der Daten an allen Station aktiv zu lassen.

Intervalllänge für Maximaldurchsatz

Da es sich bei Kundenankünften um diskrete Ereignisse handelt, kann die Erfassung des maximalen Durchsatzes, d.h. der maximalen Anzahl an Kundenankünften pro Zeiteinheit nur jeweils über explizit angegebene Intervalle erfolgen. Auf dieser Dialogseite kann die jeweils zu verwendende Intervalllänge pro Station eingestellt werden.

Aus Gründen der besseren Vergleichbarkeit ist es empfehlenswert, denselben Wert für alle Stationen zu verwenden. Weiterhin ist zu bedenken, dass sich stochastische Effekte um so stärker gegenseitig ausmitteln, je länger die Intervalle gewählt sind, d.h. kürze Intervalle führen zu tendenziell höheren maximalen Durchsätzen.
