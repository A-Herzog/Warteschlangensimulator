Quelle konfigurieren

Bei den meisten Stationen kann konfiguriert werden, wie die Verarbeitung der Kunden an diesen erfolgen soll. Bei einer Bedienstation kann z.B. eingestellt werden, wie lange die Bedienungen dauern sollen, und bei einer Kundenquelle kann eingestellt, werden wie viele Kunden und mit welchen Abständen und welchen Typs in ihr generiert werden sollen.

Im ersten Schritt konfigurieren wir die Quelle. Der Konfigurationsdialog eines Elements wird Doppelklick auf das Element geöffnet. Im Quelle-Konfigurationsdialog können folgende Einstellungen vorgenommen werden:

  • Name: Der Name der Quelle bestimmt zugleich den Typ der Kunden, die in ihr generiert werden. Hier können Sie einen beliebigen Namen eingeben oder auch den bisherigen beibehalten. Lediglich leer darf das Feld nicht sein.
  • Berechnung der Zwischenankunftszeiten: In diesem Auswahlfeld kann eingestellt werden, ob die Zwischenankunftszeiten über eine Wahrscheinlichkeitsverteilung, einen Ausdruck oder einen Zeitplan festgelegt werden sollen. In diesem Beispiel behalten wir den Wert "Wahrscheinlichkeitsverteilung" bei.
  • Zeitbasis: Bei der Bestimmung der Zwischenankunftszeiten über eine Wahrscheinlichkeitsverteilung oder einen Ausdruck kann eingestellt werden, ob der sich jeweils ergebende Zahlenwert als Anzahl an Sekunden, Minuten oder Stunden interpretiert werden soll. Auch hier kann der Vorgabewert "Sekunden" beibehalten werden.
  • Verteilung: Über die Schaltfläche "Bearbeiten" rechts über der Ansicht der Wahrscheinlichkeitsverteilung kann diese bearbeitet werden. In dem Beispielmodell soll die Exponentialverteilung beibehalten werden, aber als mittlerer Abstand zwischen zwei Ankünften sollen 50 Sekunden eingetragen werden.
  • Anzahl an Kunden pro Ankunftsereignis: Pro Ankunftsereignis können mehrere Kunden am System eintreffen (Batch-Ankünfte). In diesem Beispiel soll jedoch die Batch-Größe von 1 beibehalten werden.
  • Anzahl an Ankunftsereignis insgesamt: Die Anzahl Ankunftsereignis, die diese Quelle generiert, kann generell limitiert werden. In diesem Beispiel soll jedoch keine Begrenzung der Anzahl an Ankünften vorgenommen werden.
  • Startzeitpunkt für erste Zwischenankunftszeit: Optional kann angegeben werden, dass die Quelle erst ab einem bestimmten Zeitpunkt beginnt, Ankünfte zu generieren. In diesem Beispiel soll jedoch der Startzeitpunkt 0 beibehalten werden.

Konkrete Schritte:

  • Klicken Sie doppelt auf die Quelle:
  • Klicken Sie in dem Dialog über dem Verteilungsdiagramm rechts auf "Bearbeiten" und wählen Sie im Verteilungseditor 50 (Sekunden) als mittlere Zwischenankunftszeit. (Behalten Sie die Exponentialverteilung als Zwischenankunftszeitenverteilung bei.)
  • Schließen Sie den Verteilungseditor und den Dialog zum Bearbeiten der Quelle jeweils mit "Ok".
