Bedienstation und Ausgang verbinden

Wenn die Kante-hinzufügen-Funktion aktiviert ist, können Sie neue Kanten hinzufügen, in dem Sie nacheinander das Ausgangs- und das Zielelement der Verknüpfung anklicken.
