Abschluss der Einführung

Ihr erstes Warteschlangenmodell ist nun vollständig.

Nächste Schritte

Über die Schaltflächen Animation starten und Simulation starten in der Symbolleiste kann nun eine Animation des Modell gestartet werden oder ein kompletter Simulationslauf durchgeführt werden. Am Ende einer Simulation werden die relevanten Kenngrößen für Warte- und Bedienzeiten, Warteschlangenlängen und Bedienerauslastungen angezeigt.

Um mit dem Modell zu experimentieren, können Sie z.B. die Zwischenankunftszeiten oder die Bedienzeiten verändert oder eine Nachbearbeitungszeit hinzufügen und die Auswirkungen auf die Kenngrößen untersuchen.
