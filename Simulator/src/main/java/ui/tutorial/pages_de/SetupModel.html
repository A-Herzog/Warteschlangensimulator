Modell konfigurieren

In den vorherigen Schritten wurde eingestellt, dass im Mittel alle 50 Sekunden ein Kunde eintrifft, eine Bedienung jedoch im Mittel 80 Sekunden dauert. Des Weiteren wurde direkt im Einstellungen-Dialog der Bedienstation eine Bedienergruppe bestehend aus einem Bediener angelegt. (Und es wurde eingestellt, dass zur Bedienung eines Kunden jeweils ein Bediener notwendig ist.)

Momentan herrscht also noch eine Überlast in dem Modell vor. Daher muss die Anzahl an verfügbaren Bedienern in der Bedienergruppe noch von 1 auf 2 erhöht werden.

Einstellungen zu den Ressourcen und zu allen weiteren Aspekten, das Modell als solches betreffend, können über den Modelleigenschaften-Dialog vorgenommen werden. Dieser Dialog besitzt mehrere Registerreiter. Wir wählen den Reiter "Bediener" und erhöhen auf dieser die Anzahl an Bedienern in der vorab definierten Gruppe von 1 auf 2.

Konkrete Schritte:

  • Klicken Sie im Hauptfenster auf diese Schaltfläche:
  • Wechseln Sie in dem Dialog auf die Registerseite Bediener.
  • Klicken Sie in der Spalte Anzahl auf das Plus-Symbol, um die Anzahl von 1 auf 2 zu erhöhen.
  • Schließen Sie den Dialog mit "Ok".
