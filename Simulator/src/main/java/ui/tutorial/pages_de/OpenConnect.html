Elemente verbinden

Um die Elemente des Warteschlangenmodells zu verbinden und damit den Weg der Kunden durch das System zu beschreiben, werden sogenannte Kanten verwendet. Eine Kante hat immer ein Ausgangs- und ein Zielelement und wird durch einen Pfeil symbolisiert. Im Gegensatz zu Elementen können Kanten nicht eigenständig bestehen. Wird das Ausgangs- oder das Zielelement einer Kante gelöscht, so verschwindet auch die Kante.

Um eine Kante in dem Modell hinzuzufügen, muss die Kanten-Funktion in der vertikalen Symbolleiste links im Hauptfenster aktiviert werden. Klicken Sie dafür auf die "Kante"-Schaltfläche in der Symbolleiste links.
