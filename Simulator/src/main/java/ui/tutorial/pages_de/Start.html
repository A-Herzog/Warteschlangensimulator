Einführung

Willkommen in der interaktiven Einführung zum Warteschlangensimulator.

Dieses Tutorial hilft Ihnen dabei, ein erstes einfaches Warteschlangenmodell zu erstellen. In diesem Fenster wird dabei angezeigt, welche Schritte Sie jeweils durchführen müssen, um das Modell weiter aufzubauen. Sobald Sie die entsprechende Komponente zu dem Modell hinzugefügt haben bzw. die entsprechende Einstellung vorgenommen haben, wird die jeweils nächste Seite angezeigt.

Klicken Sie unten auf "Start", um mit der Erstellung des Modells zu beginnen.
