Setup model

In the previous steps an average inter-arrival time of 50 seconds and an average processing time of 80 seconds per client was defined. Furthermore, an operator group consisting of one operator was created strait from the setup dialog of the process station. (And was set that to one operator is needed to serve one client.)

At the moment therefore there is an overload in the model. So the number of available operators in the operator group needs to be increased from 1 to 2.

All resources and many more model related setting can be adjusted in the Model properties dialog. This dialog has several tabs. We select the tab "Operators" tab and increase the number of operators in the pre-defined operators group from 1 to 2.

Concrete steps:

  • Click in the main window on this button:
  • Change to the Operator tab in the dialog.
  • Click on the plus symbol in the Number column to increase the number of operators in the group from 1 to 2.
  • Close the dialog by clicking "Ok".
