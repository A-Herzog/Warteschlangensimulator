Setup source

At most stations, it is possible to configure how clients are to be processed. In a process station, e.g. one can setup how long the service process should last, and at a client source can be set, how many clients are to be generated, at which inter-arrival times and of which types.

The first step is to configure the source. The configuration dialog of an element can be opened by double clicking on it. The following settings can be made in the source configuration dialog:

  • Name: The name of the source also determines the type of clients that are generated in it. Here you can enter any name or keep the previous name. The field may not be empty.
  • Calculation of the inter-arrival times: This selection field can be used to specify whether the inter-arrival times should be defined by a probability distribution, an expression or a schedule. In this example we will keep the value "Probability distribution".
  • Time base: When determining the inter-arrival times by means of a probability distribution or an expression, it can be set whether the respective numerical value is to be interpreted as the number of seconds, minutes or hours.The default value "seconds" can also be retained here.
  • Distribution: By clicking "Edit" button on the right above the probability distribution the distribution can be edited. In the example model, the exponential distribution is to be maintained, but the average distance between two arrivals should be 50 seconds.
  • Number of clients per arrival event: At each arrival event multiple clients can arrive (batch arrival). In this example the batch size of 1 should be maintained.
  • Number of arrival events in total: The number of arrival events that this source generates can generally be limited. In this example no limitation of the number of arrivals is to be made.
  • Start time for the first inter-arrival time: Optionally, you can specify that the source will start generating arrivals only from a certain point in time. In this example the start time 0 is to be maintained.

Concrete steps:

  • Double click on the source:
  • In the dialog, click Edit on the right side of the distribution diagram and select 50 (seconds) as the average inter-arrival time in the distribution editor. (Keep the exponential distribution as the inter-arrival time distribution.)
  • Close the distribution editor and the dialog for editing the source with "Ok".
