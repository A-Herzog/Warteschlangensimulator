Element templates

A queueing model is essentially a special type of a flow chart. Each queueing model consists of elements and connection edges.

As a first step, the three elements from which this first queueing model is to be constructed are to be added to the model.

Templates for all available elements are located in a sidebar on the left side of the main window.

Click on the "Element" button on the left side of the program window to open the templates panel. The templates panel can be opened and closed at any time using the "Element" button.
