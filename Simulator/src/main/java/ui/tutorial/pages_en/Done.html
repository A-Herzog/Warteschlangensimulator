Next steps

You can start an animation of the model or execute a complete simulation run by clicking the buttons Start animation and Start simulation on the toolbar. At the end of a simulation, the relevant performance indicators for waiting and process times, queue lengths and operator utilization are displayed.

To experiment with the model, change the interarrival times or the process times or add a post-processing time and examine the effects on the performance indicators.
