Setup process station

At most stations, it is possible to configure how clients are to be processed. In a process station, e.g. one can setup how long the service process should last, and at a client source can be set, how many clients are to be generated, at which interarrival times and of which types.

After configuring the source, the process station has to be configured. (The exit element has no settings.) The configuration dialog of an element is opened by double-clicking on the element. The process station configuration dialog consists of several pages:

  • Processing times: Set an exponentially distributed with average duration of 80 seconds here, similar to the source. Time base (seconds) and recording the delay times as process times can be kept without any change.
  • Setup times: When serving clients of different types may be setup times between clients of different types are needed. In this example, setup times should be kept disabled.
  • Post processing times: After a client has been served, a post processing time may be necessary before the operator will be available again. In this example, post processing times should be kept disabled.
  • Waiting time tolerances: If a client has to wait too long, he may cancels waiting (because his waiting time tolerance has been exceeded). In this example, the waiting time tolerances should be kept disabled.
  • Priorities and batch sizes: On this page you can specify by which strategy clients are taken from the queue and whether batch operations are to be performed. The selection order "w" means a priorization of the clients by their waiting times, that is, using a FIFO queue. In this example, the value "w" should be maintained for the priority determination of the clients and a batch size of 1.
  • Operators: This page can be used to determine which numbers of which types of operators are required to serve a client. Click "Add needed operators group" and confirm with "OK" to create a group consisting of one operator and add it as a necessary operator group.
  • Costs: On this page you can optionally set costs for the operations.

Concrete steps:

  • Double click on the process station:
  • On the "Processing times" page click on Edit above the distribution diagram and select 80 (seconds) as the average operating time.
  • On the "Operators" page, click "Add needed operators group", and close the dialog by clicking on "OK".
